## 1. 价值定位

### 1.1 解决的问题
酒店能耗成本占总运营成本15-25%，面临严峻挑战：

**能耗成本居高不下**
- 电费占能耗成本60-70%，夏季空调费用惊人
- 客房空置期间仍维持恒温，浪费严重
- 老旧设备能效低，改造资金压力大

**能耗管理粗放**
- 缺乏实时监测，月底账单" surprise"
- 设备运行凭经验设定，未基于实际需求
- 公共区域照明、空调常开，无人管控

**节能措施难量化**
- 投入节能设备后，难以验证ROI
- 节能效果随季节、入住率波动
- 缺乏数据支撑，管理层决策困难

### 1.2 核心价值
**"让每一度电都产生价值，实现' invisible savings '"**

- **智能调控**：AI实时优化HVAC运行，节能15-25%
- **削峰填谷**：动态电价管理，降低用电成本20%+
- **预测优化**：基于入住率预测提前调整，避免浪费
- **可视透明**：能耗数据实时可视化，管理心中有数

### 1.3 量化收益

| 指标 | 传统模式 | 智能能源管理 | 节省幅度 |
|------|---------|-------------|---------|
单位面积能耗 | 200 kWh/m²/年 | 150 kWh/m²/年 | **-25%** |
能源成本占比 | 20-25% | 15-18% | **-5-7pp** |

#### 模块2: AI负荷预测

**预测模型**
```python
class LoadPredictor:
    """基于LSTM的负荷预测模型"""
    
    def __init__(self):
        self.model = self.build_lstm_model()
    
    def build_lstm_model(self):
        model = Sequential([
            LSTM(128, return_sequences=True, input_shape=(24, 10)),
            Dropout(0.2),
            LSTM(64, return_sequences=False),
            Dropout(0.2),
            Dense(32, activation='relu'),
            Dense(24)  # 预测未来24小时
        ])
        model.compile(optimizer='adam', loss='mse')
        return model
    
    def predict_next_24h(self, features):
        """
        输入特征:
        - 历史负荷 (过去7天每小时)
        - 入住率预测 (来自PMS)
        - 天气预报 (温度、湿度)
        - 日期特征 (工作日/周末/节假日)
        - 特殊事件 (会议/宴会)
        
        输出: 未来24小时每小时负荷预测
        """
        prediction = self.model.predict(features)
        return prediction
```

**预测应用场景**

```
场景1: 空调预冷优化
```

```python
class HVACOptimizer:
    """基于强化学习的HVAC优化控制"""
    
    def __init__(self):
        self.rl_agent = DQNAgent(
            state_dim=20,  # 状态维度
            action_dim=10,  # 动作维度
            learning_rate=0.001
        )
    
    def get_state(self):
        """获取当前状态"""
        return {
            'indoor_temp': get_indoor_temp(),
            'outdoor_temp': get_outdoor_temp(),
            'occupancy': get_occupancy(),
            'power_load': get_current_load(),
            'time_of_day': get_hour(),
            'setpoint': get_current_setpoint()
        }
    
    def take_action(self, state):
        """AI决策动作"""
        action = self.rl_agent.select_action(state)
        
        # 动作空间
        actions = {
            0: '维持当前设定',
            1: '设定温度+0.5℃',
            2: '设定温度-0.5℃',
            3: '风速提高一档',
            4: '风速降低一档',
            5: '开启节能模式',
            6: '关闭节能模式',
            7: '提前30分钟启动',
            8: '延迟30分钟启动',
            9: '切换至外气冷房'
        }
        
        return actions[action]
    
    def calculate_reward(self, energy_saved, comfort_score):
        """计算奖励函数"""
        # 节能与舒适的平衡
        reward = energy_saved * 0.6 + comfort_score * 0.4
        return reward
```

#### 模块4: 峰谷电价优化

**电价结构分析**
```
以某市商业电价为例:
├── 峰时 (08:00-11:00, 13:00-17:00, 19:00-21:00)
│   电价: ¥1.2/kWh (基准)
├── 平时 (07:00-08:00, 11:00-13:00, 17:00-19:00, 21:00-23:00)
│   电价: ¥0.8/kWh (-33%)
└── 谷时 (23:00-07:00)
    电价: ¥0.4/kWh (-67%)
```

**削峰填谷策略**
```python
class PeakShavingOptimizer:
    """峰谷电价优化调度"""
    
    def optimize_schedule(self, load_forecast, price_schedule):
        """
        优化策略:
        1. 谷时预蓄冷/热
        2. 峰时削减非必要负荷
        3. 可转移负荷调整时段
        """
        
        schedule = {}
        
        for hour in range(24):
            price = price_schedule[hour]
            
            if price == 'valley':  # 谷时
                # 充分利用低价电
                schedule[hour] = {
                    'chiller': 'full_load',  # 冷水机组满负荷蓄冷
                    'water_heater': 'on',     # 热水器加热
                    'laundry': 'schedule',    # 安排洗衣
                    'charging': 'ev_charging' # 电动车充电
                }
            
            elif price == 'peak':  # 峰时
                # 削减负荷
                schedule[hour] = {
                    'chiller': 'reduce_load',  # 降低负荷(利用蓄冷)
                    'non_essential': 'off',     # 关闭非必要设备
                    'lighting': 'dim_20'        # 照明调暗20%
                }
        
        return schedule
```

#### 模块5: 能耗异常检测

**异常类型**
| 异常类型 | 检测方法 | 示例 |
|---------|---------|------|
**突然跳变** | 阈值检测 | 某回路功率突然翻倍 |
**持续偏高** | 统计异常检测 | 空调能耗比历史同期高30% |
**夜间异常** | 时间序列分析 | 凌晨2点客房区高负荷 |
**比例失衡** | 相关性分析 | 制冷量与电耗不匹配 |
**设备故障** | 特征识别 | 冷水机组COP骤降 |

**检测算法**
```python
class AnomalyDetector:
    """能耗异常检测"""
    
    def __init__(self):
        self.isolation_forest = IsolationForest(contamination=0.05)
        self.statistical_model = None
    
    def detect_anomalies(self, data):
        """
        多算法融合检测
        """
        anomalies = []
        
        # 1. 统计异常检测 (3-sigma规则)
        mean = np.mean(data)
        std = np.std(data)
        statistical_anomalies = np.where(np.abs(data - mean) > 3 * std)[0]
        
        # 2. Isolation Forest检测
        isolation_scores = self.isolation_forest.fit_predict(data.reshape(-1, 1))
        isolation_anomalies = np.where(isolation_scores == -1)[0]
        
        # 3. 时间序列异常 (LSTM reconstruction error)
        lstm_errors = self.lstm_reconstruction_error(data)
        lstm_anomalies = np.where(lstm_errors > threshold)[0]
        
        # 融合结果
        all_anomalies = set(statistical_anomalies) | set(isolation_anomalies) | set(lstm_anomalies)
        
        return list(all_anomalies)
```
