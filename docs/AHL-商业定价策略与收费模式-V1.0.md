## 1. 定价策略总览

### 1.1 核心定价理念

```yaml
分层定价:
  理念: "按需付费，用多少付多少"
  逻辑: 基于酒店规模、业态复杂度、功能需求分层
  优势: 降低入门门槛，覆盖全市场

模块化收费:
  理念: "像乐高一样自由组合"
  逻辑: 必配基础包 + 选配AGENT + 按量SKILL
  优势: 灵活适配，避免功能浪费

效果导向:
  理念: "共赢共享，风险共担"
  逻辑: 基础订阅 + 效果分成
  优势: 降低客户决策风险，提升LTV
```

### 1.2 定价矩阵概览

| 酒店规模 | 房间数 | 基础年费 | AGENT选配 | 预估总投入 |

## 2. 收费模式详解

### 2.1 三层收费结构

```
总费用 = 基础订阅费 + AGENT授权费 + 增值服务/效果分成费

┌─────────────────────────────────────────────────────────┐
│                   AHL收费结构                            │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  第一层: 基础订阅费 (必选)                                │
│  ├─ 核心运营AGENT (必配)                                 │
│  ├─ 平台基础设施使用费                                    │
│  └─ 基础技术支持                                         │
│                                                          │
│  第二层: AGENT授权费 (按需选配)                           │
│  ├─ 营销增长AGENT                                        │
│  ├─ 收益管理AGENT                                        │
│  ├─ 工程维保AGENT                                        │
│  ├─ 安防监控AGENT                                        │
│  ├─ 宴会销售AGENT                                        │
│  ├─ 财务风控AGENT                                        │
│  └─ 人力行政AGENT                                        │
│                                                          │
│  第三层: 增值服务/效果分成 (可选)                          │
│  ├─ 新客开发效果分成 (3-5%)                              │
│  ├─ 能耗优化收益分成 (10-15%)                            │
│  ├─ 专属客户成功经理                                      │
│  └─ 定制化开发服务                                       │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### 2.2 收费模式选项

| 模式 | 适用客户 | 收费方式 | 优势 |
|------|---------|---------|------|
**纯订阅制** | 预算充足、求稳型 | 固定年费 | 成本可控、简单透明 |
**订阅+分成** | 追求效果、愿意共担风险 | 基础费+效果分成 | 风险共担、激励对齐 |
**效果付费** | 预算紧张、效果导向 | 纯效果分成 | 零 upfront、按结果付费 |
**买断制** | 大型集团、数据敏感 | 一次性买断+年费维护 | 完全控制、长期成本低 |

    Year1: ¥7.30万

    Year4: ¥6.98 × 0.85 × 1.15 = ¥6.82万
