## 一、工程部SOP资料

### 1.1 设备维护保养SOP

#### 来源：酒店管理网 - 工程部设备维护保养制度
**核心要点**:

**日常保养（Daily Maintenance）**:
- 早班检查（8:00-9:00）:
  - 锅炉房设备运行状态检查
  - 空调主机运行参数记录
  - 水泵房水位/压力检查
  - 电梯运行状态确认
  - 发电机待机状态检查
  
- 巡检路线（每2小时）:
  - 配电室 → 空调机房 → 锅炉房 → 水泵房 → 电梯机房
  - 记录表：设备运行日志

**周期性保养（Periodic Maintenance）**:

| 设备 | 保养周期 | 保养内容 | 责任人 |
|------|---------|---------|--------|
| 中央空调 | 月度 | 滤网清洗、冷媒检查、皮带调整 | 空调技工 |
| 锅炉 | 季度 | 水垢清理、安全阀检测、燃烧器调试 | 锅炉工 |
| 电梯 | 半月 | 导轨润滑、安全装置测试、轿厢检查 | 电梯维保 |
| 发电机 | 周度 | 空载试运行、燃油检查、电瓶维护 | 电工 |
| 水泵 | 月度 | 轴承润滑、密封检查、电机绝缘测试 | 管工 |

**年度大修（Annual Overhaul）**:
- 空调系统：冷媒更换、压缩机检修、管道清洗
- 电气系统：变压器检测、配电柜维护、接地测试
- 给排水：管道除锈、阀门更换、水箱清洗消毒

#### 来源：百度文库 - 酒店工程部SOP手册

**紧急维修响应流程**:
```
报修接收 → 10分钟内响应 → 现场勘查 → 维修方案 → 实施维修 → 验收确认 → 记录归档
```

**维修分级**:
- A级（紧急）：停水停电、电梯困人、燃气泄漏 → 15分钟内到场
- B级（重要）：空调故障、热水中断、门锁损坏 → 30分钟内到场
- C级（一般）：灯具更换、五金维修、小修补 → 2小时内到场

## 二、安保部SOP资料

### 2.1 消防安全SOP

#### 来源：消防标准化管理网 - 酒店消防安全操作规程

**消防控制室值班制度**:
- 24小时双人值班
- 每2小时巡查记录
- 火警信号3分钟内现场确认
- 监控录像保存30天以上

**日常防火巡查 checklist**:
- [ ] 疏散通道畅通无阻
- [ ] 安全出口标志完好
- [ ] 灭火器压力正常（绿区）
- [ ] 消火栓水带完好
- [ ] 应急照明功能正常
- [ ] 防火门关闭状态
- [ ] 客房内无违规电器
- [ ] 厨房油烟管道清洁
- [ ] 配电室温湿度正常
- [ ] 燃气管道无泄漏

**火灾应急处理流程**:
```
发现火情 → 按下手动报警按钮 → 通知消防控制室 → 组织人员疏散
    ↓
初期扑救（使用灭火器/消火栓）→ 引导消防队 → 保护现场 → 事后调查
```

#### 来源：保安协会 - 酒店安保巡逻标准

**巡逻路线与频次**:
- 大堂/前台：每小时1次
- 客房楼层：每2小时1次
- 停车场：每2小时1次
- 后勤区域：每4小时1次
- 外围周界：每4小时1次

**巡逻检查要点**:
- 可疑人员识别与询问
- 门窗锁闭状态检查
- 消防设施设备检查
- 监控盲区现场确认
- 安全隐患排查记录

### 2.2 监控中心SOP

**监控操作规范**:
- 重点监控区域：大堂、前台、收银处、电梯、出入口
- 异常行为识别：徘徊、尾随、争执、可疑物品
- 录像调取流程：申请→审批→调取→登记→归还
- 设备故障报修：发现故障→记录→报修→跟进

## 三、财务部SOP资料

### 3.1 前台收银SOP

#### 来源：酒店财务管理网 - 前台收银操作规范

**入住收银流程**:
1. 核对预订信息（房型/房价/入住天数）
2. 收取押金（现金/刷卡/预授权）
3. 开具押金单（一式两联）
4. 系统录入押金信息
5. 将押金单放入客账袋

**押金标准**（参考）:
| 房型 | 现金押金 | 信用卡预授权 |
|------|---------|-------------|
| 标准间 | 500元 | 1000元 |
| 商务间 | 800元 | 1500元 |
| 套房 | 1500元 | 3000元 |

**退房结账流程**:
1. 收回房卡
2. 查房确认（电话查房或系统确认）
3. 核对迷你吧消费
4. 计算总消费金额
5. 退还押金/结算差额
6. 开具发票
7. 系统退房操作
8. 交还证件

**备用金管理**:
- 前台备用金：每班次5000-10000元
- 交接班盘点：当面清点、签字确认
- 长短款处理：当日上报、填写说明、逐级审批
- 大额现金：及时投库、不得过夜

### 3.3 成本控制SOP

#### 来源：酒店成本管理实务

**客房成本核算**:
```
客房成本 = 固定成本 + 变动成本

固定成本：
- 客房折旧
- 客房人员工资
- 客房保险

变动成本：
- 布草洗涤费
- 一次性用品
- 清洁用品
- 迷你吧商品成本
- 水电能耗
```

**布草洗涤成本控制**:
- 洗涤量统计（每日/每周/每月）
- 洗涤单价谈判与合同管理
- 洗涤质量验收标准
- 报损率控制（<3%）

**一次性用品成本管控**:
- 标准配置量（牙刷/牙膏/梳子/拖鞋等）
- 补充频率控制
- 库存周转天数（<30天）
- 单房耗用品成本核算

**能耗成本控制**:
- 电力：分区域计量、峰谷电价利用
- 水费：中水回用、节水器具
- 燃气：锅炉效率监测
- 月度能耗分析报告
