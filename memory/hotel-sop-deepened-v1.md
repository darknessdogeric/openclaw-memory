## 一、前厅部SOP（深化版）

### 1.1 预订管理SOP

#### 1.1.1 电话预订流程

**目的**: 规范电话预订接待流程，确保信息准确录入

**适用范围**: 前台/预订部员工

**流程步骤**:

```
步骤1: 三声内接听电话
    ↓ 问候语:"您好，XX酒店预订部，很高兴为您服务"
步骤2: 询问客人需求
    ↓ 入住日期、离店日期、房型需求、房间数量
步骤3: 查询房态并报价
    ↓ 确认当日可售房型及价格
步骤4: 确认预订信息
    ↓ 房型、房价、入住人姓名、联系方式、到店时间
步骤5: 询问特殊需求
    ↓ 无烟房/高层/靠近电梯/婴儿床等
步骤6: 确认预订
    ↓ 告知预订号、取消政策、保留时间
步骤7: 录入系统并发送确认短信/邮件
```

**关键控制点**:
| 控制点 | 标准 | 检查方式 |
|--------|------|----------|
| 接听速度 | 3声内 | 电话录音抽查 |
| 信息完整率 | 100% | 系统录入检查 |
| 报价准确率 | 100% | 财务审计 |
| 确认发送率 | 100% | 系统日志 |

**相关表单**:
- 《预订确认单》
- 《每日预订汇总表》
- 《No-Show跟踪表》

---

#### 1.1.2 网络预订处理

**来源渠道**: OTA(携程/美团/Booking) + 官网 + 小程序

**处理流程**:

**OTA订单**:
```
订单接收(自动) → 房态确认(30分钟内) → 录入PMS → 发送确认
    ↑                                           ↓
    └──────── 拒单/申请关房(满房时) ←──────────┘
```

**关键操作**:
1. **接单时限**: 30分钟内必须确认或拒绝
2. **满房处理**: 立即申请关房，联系OTA客服
3. **特殊订单**: 留意备注栏（加床、无烟、连通房等）
4. **价格核对**: 确保OTA价格与PMS一致
5. **到店确认**: 提前1天电话确认到店时间

**OTA预订检查清单**:
- [ ] 订单已录入PMS
- [ ] 房型和房价正确
- [ ] 入住人姓名完整
- [ ] 特殊需求已备注
- [ ] 确认短信已发送
- [ ] 预付订单已标记

### 1.2 入住接待SOP

#### 1.2.1 散客入住流程

**准备工作** (班前15分钟):
- [ ] 检查仪容仪表
- [ ] 准备房卡、登记单、押金单
- [ ] 核对当日预订情况
- [ ] 检查备用金
- [ ] 了解当日VIP/特殊客人

**接待流程**:

```
客人到达
    ↓
微笑问候:"您好，欢迎光临XX酒店"
    ↓
询问是否有预订
    ↓
有预订 → 核对预订信息 → 查询预排房
无预订 → 查询房态 → 推荐房型
    ↓
证件登记(身份证/护照扫描)
    ↓
收取押金(现金/刷卡/预授权)
    ↓
制作房卡 + 准备早餐券/康乐券
    ↓
介绍酒店设施和服务
    ↓
祝客人入住愉快，指引电梯方向
    ↓
录入系统，归档资料
```

**3分钟入住标准**:
| 步骤 | 时间 | 动作标准 |
|------|------|----------|
| 问候+确认预订 | 30秒 | 微笑、眼神接触 |
| 证件登记 | 60秒 | 扫描、核对、上传公安系统 |
| 押金收取 | 45秒 | 开单、收款/刷卡、确认 |
| 房卡制作 | 30秒 | 写卡、测试、装卡套 |
| 设施介绍 | 45秒 | 早餐时间/地点、WIFI、退房时间 |
| 送别 | 30秒 | 祝福语、指引方向 |

**证件登记规范**:
- 境内客人: 二代身份证原件
- 境外客人: 护照原件
- 港澳台: 回乡证/台胞证
- 未成年人: 户口本+监护人身份证
- 上传公安系统: 入住后1小时内

**押金标准**:
| 房型 | 现金押金 | 信用卡预授权 | 支付宝/微信 |
|------|---------|-------------|------------|
| 经济房 | 300元 | 500元 | 500元 |
| 标准间 | 500元 | 800元 | 800元 |
| 商务间 | 800元 | 1200元 | 1200元 |
| 套房 | 1500元 | 3000元 | 3000元 |

#### 1.2.2 VIP入住流程

**VIP等级定义**:
| 等级 | 标准 | 接待人员 |

**VIP入住流程**:
```

    ↓
引领至房间(非柜台办理) → 介绍房间设施
    ↓
询问特殊需求 → 10分钟内回访
```

**VIP在住关注**:
- 每日问候(电话/当面)
- 快速响应需求(5分钟内)
- 离店前询问入住体验
- 赠送小礼品

### 1.3 退房结账SOP

#### 1.3.1 标准退房流程

**退房时间**: 12:00(标准) / 14:00(会员)

**流程**:
```
客人提出退房
    ↓
收回房卡
    ↓
电话查房(或系统确认)
    ↓
核对迷你吧消费
    ↓
打印账单请客人确认
    ↓
结算(退还押金/补差价)
    ↓
开具发票
    ↓
系统退房
    ↓
送别:"感谢您的入住，期待再次光临"
    ↓
整理账单资料归档
```

**查房标准** (3分钟内完成):
- 客房部接到查房电话立即执行
- 检查项目: 迷你吧/布草/设施/遗留物品
- 有消费: 报消费项目和金额
- 无消费: 报"房间正常"

**发票规范**:
- 抬头必须与付款人一致
- 发票内容: 住宿费/会务费/餐费
- 金额与实际消费一致
- 当日可作废，隔月需红冲

**退房高峰应对** (9:00-11:00):
1. 增开退房窗口
2. 提前电话查房(预知退房的)
3. 引导自助退房(会员)
4. 快速退房服务(免查房)

### 1.5 行李服务SOP

#### 1.5.1 行李寄存

**寄存流程**:
```
客人要求寄存 → 询问寄存物品类型
    ↓
填写《行李寄存卡》(一式两联)
    ↓
上联给客人，下联挂行李上
    ↓
行李放入行李房，按日期分区
    ↓
登记《行李寄存登记表》
```

**寄存规定**:
- 贵重物品、易碎品、危险品不予寄存
- 寄存期限: 一般30天，超期无人认领按无主处理
- 凭寄存卡领取，无卡需核实身份
- 代领需寄存人授权+代领人证件

#### 1.5.2 行李送房/离店

**送房服务**:
- 收到行李后10分钟内送至房间
- 与客人确认行李件数
- 摆放整齐(衣柜/行李架)

**离店服务**:
- 按客人要求时间送行李下楼
- 协助装车
- 与客人确认件数

## 二、客房部SOP（深化版）

### 2.1 客房清洁SOP

#### 2.1.1 走客房清洁（退房清洁）

**准备工作**:
- [ ] 准备清洁车(布草/用品/清洁剂)
- [ ] 检查清洁工具完好
- [ ] 查看房态表确认房间状态

**清洁流程** (标准时间: 30-40分钟/间):

**第一步: 进房程序** (2分钟)
```
按门铃3下，报"客房服务"
    ↓
等待3秒，无反应后刷卡进入
    ↓
开窗通风(如需要)
    ↓
打开窗帘
    ↓
检查房间是否有遗留物品
    ↓
关灯关空调(节能)
```

**第二步: 撤除脏布草和垃圾** (5分钟)
```
撤除床上布草(床单/被套/枕套)
    ↓
撤除卫生间布草(毛巾/地巾/浴巾)
    ↓
收集垃圾(清空垃圾桶)
    ↓
分类放置(布草袋/垃圾袋)
```

**第三步: 铺床** (8分钟)
```
铺床单(包角45度，中线居中)
    ↓
套被套(四角对齐，开口朝床尾)
    ↓
套枕套(饱满均匀，开口朝内)
    ↓
床尾巾(如配置)
    ↓
检查床面平整
```

**铺床质量标准**:
- 床单: 中线居中，四边下垂均匀(25-30cm)
- 包角: 45度，紧实整齐
- 被套: 平整无皱，被芯均匀分布
- 枕头: 饱满挺括，开口向内

**第四步: 卫生间清洁** (10分钟)
```
镜面 → 台面 → 水龙头 → 马桶(由外向内)
    ↓
浴缸/淋浴间(墙面→地面→地漏)
    ↓
补充客用品(毛巾/洗漱用品/卷纸)
    ↓
擦干水迹
    ↓
检查排水通畅
```

**第五步: 家具电器清洁** (8分钟)
```
家具表面(从里到外，从上到下)
    ↓
电器(电视/空调/灯具/开关)
    ↓
窗台/窗帘
    ↓
地面吸尘(从里到外)
```

**第六步: 补充客用品** (5分钟)
```
床上: 拖鞋(床尾)
    ↓
迷你吧: 核对补充饮料食品
    ↓
写字台: 便签/铅笔/茶包/矿泉水
    ↓
卫生间: 洗漱套装/卷纸/毛巾
    ↓
衣柜: 衣架/洗衣袋/鞋拔/拖鞋
```

**第七步: 最后检查** (2分钟)
```
环视房间，检查是否有遗漏
    ↓
检查设施设备是否正常
    ↓
调节空调温度(22-24℃)
    ↓
关闭门窗
    ↓
关灯(除廊灯外)
    ↓
锁门
    ↓
登记出房时间
```

**清洁检查表**:
| 项目 | 标准 | 检查方式 |
|------|------|----------|
| 床品 | 无毛发、无污渍、平整 | 目视+触摸 |
| 卫生间 | 无水迹、无异味、五金光亮 | 目视 |
| 地面 | 无垃圾、无毛发、无污渍 | 目视 |
| 家具 | 无灰尘、无水迹 | 手摸 |
| 物品摆放 | 按标准位置摆放 | 对照标准图 |
| 空气 | 清新无异味 | 嗅觉 |

#### 2.1.2 住客房清洁（日常整理）

**敲门程序**:
```
按门铃3下，报"客房服务，请问可以打扫房间吗"
    ↓
等待客人回应
    ↓
如客人开门: 询问是否方便打扫，如不方便记录时间再访
    ↓
如无人应答: 按进房程序进入
    ↓
如房间有人: 询问需求(简单整理/深度清洁)
```

**住客房整理标准** (15-20分钟/间):
- 不随意移动客人物品
- 不翻看客人文件/行李
- 重点清洁卫生间
- 补充客用品
- 收走垃圾
- 简单整理床铺

**敏感物品处理**:
- 现金、首饰: 不触碰，不移动
- 药物: 不触碰
- 文件资料: 不翻看
- 食品: 不过期不处理

- 使用寿命到期(床单1年/毛巾6个月)

### 2.4 迷你吧管理SOP

#### 2.4.1 迷你吧补充流程

#### 5.4.1 火灾应急预案

**应急组织架构**:

- 疏散组: 客房部+前厅部

- 救护组: 餐饮部+人事部
- 通讯组: 总机+前厅部
- 后勤组: 财务部+采购部

**火灾应急流程**:
```
发现火情 → 按下手动报警按钮 → 通知消防控制室
    ↓
初期扑救(使用灭火器/消火栓)
    ↓
确认无法控制 → 启动疏散警报
    ↓
拨打119 → 通知总经理 → 启动应急预案
    ↓
组织人员疏散(优先老弱病残)
    ↓
引导消防队 → 协助灭火 → 清点人数
    ↓
保护现场 → 事后调查 → 恢复运营
```

**疏散要点**:
- 使用消防楼梯，禁止使用电梯
- 弯腰低姿，湿毛巾捂口鼻
- 按照疏散指示方向撤离
- 到指定集合点集合
- 清点人数，报告总指挥

    ↓

    ↓

    ↓

```

    ↓
疏散周边人员 → 封锁区域
    ↓
报告总经理 → 报警
    ↓
配合警方排爆 → 事后检查
```

### 6.3 成本控制SOP

#### 6.3.1 餐饮成本控制

**成本核算公式**:
```
食品成本率 = (期初库存 + 本期采购 - 期末库存) / 食品销售收入 × 100%
```

**成本控制标准**:
| 餐饮类型 | 成本率标准 | 控制措施 |
|----------|------------|----------|
| 中餐 | 35-40% | 标准菜谱、定量采购 |
| 西餐 | 30-35% | 标准菜谱、定期盘点 |
| 自助餐 | 40-45% | 预估人数、控制出菜 |
| 宴会 | 35-40% | 标准菜单、合同锁定 |

**采购验收**:
- 三人验收(采购+厨房+财务)
- 核对数量、质量、价格
- 不合格品退货
- 当日入库登记

---

#### 6.3.2 客房用品成本控制

**单房成本标准**:
| 项目 | 成本标准 | 控制措施 |
|------|----------|----------|
| 一次性用品 | 15-25元/间夜 | 标准配置、避免浪费 |
| 布草洗涤 | 8-12元/间夜 | 控制更换频率 |
| 清洁用品 | 3-5元/间夜 | 定量使用 |
| 迷你吧商品 | 按实际销售 | 及时补充、防损耗 |

**月度成本分析**:
- 实际成本 vs 预算
- 同比/环比分析
- 异常成本调查
- 改进措施制定

### 7.2 动态定价SOP

#### 7.2.1 定价决策矩阵

| 预订进度 | <60% | 60-80% | 80-95% | >95% |
|----------|------|--------|--------|------|
| **>7天** | 促销价 | 基础价 | 溢价10% | 关房 |
| **4-7天** | 基础价-10% | 基础价 | 溢价20% | 溢价30% |
| **1-3天** | 基础价-20% | 基础价 | 溢价30% | 溢价50% |
| **当日** | 基础价-30% | Walk-in价 | 溢价50% | 满房 |

#### 7.2.2 价格调整触发条件

**涨价触发**:
- 预订进度超前(>10%)
- 竞争对手涨价
- 大型活动/展会
- 节假日
- 团队取消

**降价触发**:
- 预订进度滞后(>10%)
- 竞争对手降价
- 淡季促销
- 当日低出租率

### 7.3 渠道管理SOP

#### 7.3.1 OTA渠道管理

**携程**:
- 挂牌等级: 争取特牌/金牌
- 参与活动: 今日特惠、限时抢购
- 价格策略: 与官网价差<5%
- 点评管理: 回复率100%

**美团**:
- 适合: 本地客源、年轻客群
- 促销: 天天特价、连住优惠
- 价格: 可比携程略低

**Booking/Agoda**:
- 适合: 入境客源
- 提前期: 更长(30-90天)
- 取消政策: 更灵活

#### 7.3.2 渠道价格一致性

**价格层级**:
```
官网直销价(最低) < OTA价格(高5-10%) < 协议价(高10-15%) < Walk-in(高20%)
```

**防止串货**:
- 不同渠道不同房型命名
- 不同渠道不同礼遇(早餐/WIFI)
- 监控OTA低价(使用价格监控工具)

## 十、通用SOP与应急预案

### 10.1 宾客投诉处理SOP

#### 10.1.1 投诉处理流程

**LEARN模型**:
- **L**isten(倾听): 不打断，让客人说完
- **E**mpathize(共情): 表达理解"我理解您的感受"
- **A**pologize(道歉): 为造成的不便道歉
- **R**esolve(解决): 提出解决方案，征得同意
- **N**otify(通知): 告知处理结果，跟进满意度

**处理时限**:
| 投诉类型 | 响应时间 | 解决时间 |
|----------|----------|----------|
| 轻微投诉 | 5分钟 | 当场解决 |
| 一般投诉 | 10分钟 | 30分钟内 |
| 重大投诉 | 立即 | 2小时内 |
| 危机投诉 | 立即 | 总经理亲自处理 |

**投诉记录表**:
| 日期 | 房号 | 投诉人 | 投诉内容 | 处理人 | 处理结果 | 客人满意度 | 预防措施 |
|------|------|--------|----------|--------|----------|------------|----------|
| | | | | | | | |

### 10.2 媒体应对SOP

#### 10.2.1 媒体接待流程

**接待原则**:
- 统一口径，未经授权不接受采访
- 礼貌接待，记录媒体信息
- 立即报告总经理/公关部

**应对流程**:
```
媒体到访 → 礼貌接待到会议室
    ↓
询问采访目的 → 记录媒体名称/记者姓名/联系方式
    ↓
报告总经理 → 未经授权不发表任何言论
    ↓
指定发言人接受采访
```

**禁止事项**:
- 员工私自接受采访
- 评论竞争对手
- 透露经营数据
- 承诺无法兑现的事项

### 10.3 突发公共卫生事件应急预案

#### 10.3.1 传染病防控

**预防措施**:
- 公共区域消毒(每日2次)
- 员工健康监测(每日测温)
- 防疫物资储备(口罩/消毒液)

**发现疑似病例**:
- 立即隔离
- 报告疾控中心
- 配合流调
- 密切接触者隔离观察
- 终末消毒
