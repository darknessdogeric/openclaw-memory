## 四、使用示例

### 示例1: 读取网页
```python
from b166er_crawler import B166ERCrawler

crawler = B166ERCrawler()

# 读取网页 (自动选择最佳工具)
content, method = crawler.read_url("https://www.marriott.com/about.mi")

print(f"Content length: {len(content)}")
```

### 示例2: 搜索信息
```python
# 搜索品牌SOP
results, engine = crawler.search("Marriott brand standards", max_results=5)

for r in results:
    print(f"- {r['title']}")
    print(f"  {r['url']}")
```

### 示例3: 批量处理
```python
urls = [
    "https://example1.com",
    "https://example2.com",
    "https://example3.com"
]

for url in urls:
    content, method = crawler.read_url(url)
    if content:
        print(f"✓ {url} | {method} | {len(content)} chars")
    else:
        print(f"✗ {url} | Failed")
```
