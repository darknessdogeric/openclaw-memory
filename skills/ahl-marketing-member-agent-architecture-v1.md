**SKILL规模**: 营销18个 + 会员15个 = 33个细分SKILL

## 一、总体架构

### 1.1 双AGENT架构图

```
AHL-B端AI运营官
    │
    ├── 【营销AGENT】(Marketing Agent)
    │   │   定位: 获客、转化、品牌、渠道、活动
    │   │
    │   ├── 【品牌策略层】3个SKILL
    │   ├── 【获客渠道层】6个SKILL  
    │   ├── 【转化运营层】5个SKILL
    │   └── 【活动 campaign 层】4个SKILL
    │
    └── 【会员AGENT】(Member/CRM Agent)
        │   定位: 会员全生命周期、CRM、忠诚度、LTV最大化
        │
        ├── 【会员体系层】4个SKILL
        ├── 【生命周期层】5个SKILL
        ├── 【忠诚度引擎层】3个SKILL
        └── 【CRM数据层】3个SKILL

总计: 2个AGENT + 33个细分SKILL
```

### 1.2 与收益管理AGENT的关系

```
营销AGENT ─────┐
               ├──→ 【预订转化】──→ 收益管理AGENT
会员AGENT ─────┘       ↑
                        │
                   动态定价/库存优化
                        │
                   反馈至营销ROI优化
```

**协同逻辑**:
- 营销AGENT带来流量 → 收益管理AGENT优化转化价格
- 会员AGENT提升复购 → 收益管理AGENT优化长期LTV定价
- 收益数据反馈 → 营销AGENT优化投放ROI

## 二、营销AGENT架构 (18 SKILLs)

### 2.1 品牌策略层 (3 SKILLs)

#### SKILL-MKT-01 品牌定位引擎 (Brand Positioning Engine)

**功能定位**: 酒店品牌差异化定位与价值主张设计

**核心能力**:
```
输入: 市场数据 + 竞品分析 + 酒店资源禀赋
    ↓
处理: 
  - STDP模型分析 (Segmentation/Targeting/Differentiation/Positioning)
  - 品牌人格量表
  - 价值主张画布
    ↓
输出: 品牌定位报告 + 视觉识别建议 + 传播语系
```

**应用场景**:
- 新酒店开业品牌定位
- 老酒店品牌升级
- 竞品挤压下的差异化重构
- 子品牌孵化

**对标**: 华住"美居"定位、亚朵"第四空间"定位

---

#### SKILL-MKT-02 品牌资产管理 (Brand Asset Management)

**功能定位**: 品牌一致性管理与声誉监控

**核心能力**:
- VI/VI合规检查 (Logo/色彩/字体使用)
- 品牌内容库管理 (图片/视频/文案)
- 全渠道品牌一致性监控
- 品牌声誉实时预警 (舆情监控)

**应用场景**:
- 加盟店品牌合规审查
- 品牌危机预警与应对
- 品牌资产数字化管理

---

#### SKILL-MKT-03 内容营销引擎 (Content Marketing Engine)

**功能定位**: 品牌故事生产与内容分发

**核心能力**:
- AI生成品牌故事 (创始人故事/设计理念/在地文化)
- 多模态内容生产 (图文/短视频/直播脚本)
- 内容日历智能规划

**内容类型**:

**关键指标**:

#### SKILL-MKT-05 直销渠道建设 (Direct Booking Optimization)

**功能定位**: 官网/小程序/APP直销能力提升

**核心能力**:
- 官网SEO/SEM优化
- 微信小程序转化率优化
- APP拉新与留存策略
- 私域流量池建设 (企业微信/社群)

**直销价值公式**:
```
直销ROI = (节省的OTA佣金 - 营销成本) / 营销成本
        = (15-25%房价节约 - 获客成本) / 获客成本
```

**对标**: 华住会70%+直销占比、亚朵高直销转化率

#### SKILL-MEM-09 LTV最大化 (LTV Optimization)

**功能定位**: 会员全生命周期价值最大化

**核心能力**:
- LTV预测模型
- 高价值会员识别
- 个性化权益推荐
- 增值服务交叉销售

**LTV计算公式**:
```
LTV = (平均客单价 × 入住频次 × 毛利率) × 会员生命周期 - 获客成本
```
