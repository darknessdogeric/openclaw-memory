# 维修响应SOP (Maintenance Response SOP)

> **文档编号**: ENG-SOP-002  
> **版本**: v1.0  
> **部门**: 工程部  
> **生效日期**: 2024-01-01

---

## 1. 目的

规范维修响应流程，确保设备故障及时修复，减少对客人和运营的影响。

## 2. 适用范围

适用于酒店所有报修处理，包括客房、公共区域、后台区域的设备设施维修。

## 3. 报修渠道

| 渠道 | 接收人 | 适用场景 |
|------|--------|----------|
| 电话 | 工程值班室 | 紧急报修 |
| 对讲机 | 工程人员 | 现场报修 |
| PMS系统 | 工程主管 | 客房报修 |
| 微信/钉钉 | 工程群 | 一般报修 |
| 口头 | 工程人员 | 现场发现 |

## 4. 维修分级

### 4.1 紧急维修（一级）

**定义**: 影响客人安全或重大运营

**范围**:
- 停电
- 停水
- 电梯困人
- 燃气泄漏
- 消防系统故障
- 跑水/漏水
- 门锁失效
- 空调完全失效（极端天气）

**响应时间**: 5分钟内到场
**完成时间**: 2小时内

### 4.2 重要维修（二级）

**定义**: 影响客人体验或正常运营

**范围**:
- 客房空调故障
- 电视/电话故障
- 卫浴设备故障
- 照明故障
- 网络故障
- 厨房设备故障

**响应时间**: 15分钟内到场
**完成时间**: 4小时内

### 4.3 一般维修（三级）

**定义**: 不影响运营，可延后处理

**范围**:
- 小五金更换
- 油漆修补
- 装饰修复
- 非关键设备故障

**响应时间**: 2小时内到场
**完成时间**: 24小时内

## 5. 维修流程

### 5.1 接单

**记录要素**:
- 报修时间
- 报修人/部门
- 故障地点
- 故障描述
- 联系方式
- 紧急程度

**派单**:
- 根据专业分工派工
- 通知维修人员
- 紧急时先电话通知

### 5.2 响应

**到场确认**:
- 联系报修人
- 确认具体位置
- 了解故障详情

**初步评估**:
- 判断故障原因
- 评估维修时间
- 准备工具材料

### 5.3 维修

**维修前**:
- 设置警示标志
- 保护周边区域
- 如需断电/断水，提前通知

**维修中**:
- 按规范操作
- 注意安全
- 减少噪音/灰尘
- 客房维修注意礼貌

**维修后**:
- 测试功能
- 清理现场
- 恢复原状
- 请报修人验收

### 5.4 完工

**填写工单**:
- 维修内容
- 更换材料
- 维修时间
- 故障原因
- 预防措施建议

**系统关闭**:
- PMS系统关单
- 更新设备档案
- 记录维修历史

**反馈**:
- 向报修人确认
- 感谢理解配合
- 询问满意度

## 6. 客房维修特殊要求

### 6.1 进房程序

**DND房间**:
- 不得擅自进入
- 联系前台确认
- 或推迟维修

**有人房间**:
- 敲门/按门铃
- 报身份："工程维修"
- 等待客人开门
- 说明来意

### 6.2 维修规范

**礼貌用语**:
> "您好，我是工程部的[姓名]，来为您维修[设备]。请问现在方便吗？"

**操作规范**:
- 穿鞋套（如需要）
- 工具放垫布上
- 减少噪音
- 不触碰客人物品
- 不问隐私

**完工**:
> "[设备]已经修好了，请试一下。如有问题请随时联系前台。打扰您了，再见！"

## 7. 外包维修

### 7.1 外包范围

- 电梯维修
- 消防维保
- 空调主机维修
- 锅炉维修
- 石材养护
- 杀虫灭鼠

### 7.2 外包管理

**报修**:
- 联系维保单位
- 说明故障情况
- 要求响应时间

**配合**:
- 提供必要条件
- 现场陪同（必要时）
- 监督维修质量

**验收**:
- 测试功能
- 确认修复
- 签字确认
- 保存记录

| 指标 | 标准 |
|------|------|
| 紧急维修响应 | <5分钟 |
| 重要维修响应 | <15分钟 |
| 一般维修响应 | <2小时 |
| 维修完成率 | >98% |
| 返修率 | <5% |
| 客人满意度 | >90% |
