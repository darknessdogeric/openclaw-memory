# 预防性维护SOP (Preventive Maintenance SOP)

> **文档编号**: ENG-SOP-001  
> **版本**: v1.0  
> **部门**: 工程部  
> **生效日期**: 2024-01-01

---

## 1. 目的

通过定期检查和维护，预防设备故障，延长设备寿命，确保设施正常运行。

## 2. 适用范围

适用于酒店所有机械、电气、管道等设备的预防性维护。

## 3. 维护周期

### 3.1 日常巡检 (Daily)

| 设备 | 检查内容 | 责任人 |
|------|----------|--------|
| 配电室 | 仪表读数、温度、异味 | 值班工程师 |
| 锅炉房 | 压力、水位、燃烧状况 | 值班工程师 |
| 水泵房 | 压力、运行声音、漏水 | 值班工程师 |
| 电梯 | 运行状况、报警系统 | 值班工程师 |
| 空调机房 | 温度、压力、异常声音 | 值班工程师 |

**巡检记录**:
- 每小时记录关键参数
- 异常立即报告
- 交接班签字确认

### 3.2 周度维护 (Weekly)

| 设备 | 维护内容 |
|------|----------|
| 发电机 | 空载测试（15分钟） |
| 消防泵 | 手动启动测试 |
| 排水泵 | 自动/手动切换测试 |
| 冷却塔 | 清洁、检查风扇 |
| 厨房设备 | 清洁、检查燃气管道 |

### 3.3 月度维护 (Monthly)

| 设备 | 维护内容 |
|------|----------|
| 空调滤网 | 清洗/更换 |
| 水泵轴承 | 润滑 |
| 照明系统 | 更换损坏灯具 |
| 门锁系统 | 检查、润滑 |
| 监控系统 | 清洁镜头、检查存储 |

### 3.4 季度维护 (Quarterly)

| 设备 | 维护内容 |
|------|----------|
| 变压器 | 检查、清洁 |
| 中央空调 | 专业保养 |
| 电梯 | 年检准备 |
| 消防系统 | 全面测试 |
| 泳池设备 | 检查、保养 |

### 3.5 年度维护 (Annual)

| 设备 | 维护内容 |
|------|----------|
| 高压设备 | 预防性试验 |
| 锅炉 | 内检、水压试验 |
| 电梯 | 年度检验 |
| 消防系统 | 年度检测 |
| 避雷系统 | 接地电阻测试 |

## 4. 维护工作流程

### 4.1 计划制定

**年度维护计划**:
- 每年12月制定下年度计划
- 根据设备手册和历史记录
- 考虑淡旺季安排

**月度维护计划**:
- 每月25日制定下月计划
- 分解年度任务
- 安排人力物力

### 4.2 工单执行

**工单类型**:
| 类型 | 代码 | 优先级 |
|------|------|--------|
| 预防性维护 | PM | 计划内 |
|  corrective维修 | CM | 紧急 |
| 报修 | WR | 按紧急度 |

**工单流程**:
```
生成工单 → 分配任务 → 准备材料 → 执行维护 → 记录结果 → 关闭工单
```

### 4.3 维护记录

**记录内容**:
- 维护日期
- 维护内容
- 更换配件
- 维护人员
- 下次维护日期
- 异常情况

**档案管理**:
- 一机一档
- 电子+纸质双备份
- 保存至设备报废后2年

| 变频技术 | 20-30% |
| 智能控制 | 10-20% |
| 热回收 | 15-25% |

## 7. 应急维修

### 7.1 响应时间

| 级别 | 定义 | 响应时间 | 完成时间 |
|------|------|----------|----------|
| 紧急 | 影响安全/营业 | 5分钟 | 2小时 |
| 重要 | 影响客人体验 | 15分钟 | 4小时 |
| 一般 | 不影响运营 | 2小时 | 24小时 |

### 7.2 应急处理

**停电**:
1. 启动发电机
2. 检查原因
3. 联系供电局
4. 恢复供电后检查设备

**停水**:
1. 启动备用水箱
2. 检查原因
3. 联系水务
4. 恢复后检查水质

**设备故障**:
1. 停机断电
2. 设置警示
3. 维修或联系厂家
4. 测试后恢复

## 9. 安全管理

### 9.1 作业安全

**高处作业**:
- 持证上岗
- 佩戴安全带
- 设置警示
- 专人监护

**电气作业**:
- 断电作业
- 挂牌上锁
- 绝缘工具
- 双人作业

**动火作业**:
- 动火证
- 清理易燃物
- 配备灭火器
- 专人监护

### 9.2 化学品管理

- 分类存放
- 标识清楚
- 通风良好
- 防泄漏

## 10. 质量标准

| 指标 | 目标 |
|------|------|
| 设备完好率 | >98% |
| 故障响应时间 | <15分钟 |
| 计划维护完成率 | >95% |
| 能耗同比降低 | 3-5% |
| 安全事故 | 0 |
