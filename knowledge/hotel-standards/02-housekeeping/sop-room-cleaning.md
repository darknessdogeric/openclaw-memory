# 客房清洁SOP (Room Cleaning SOP)

> **文档编号**: HK-SOP-001  
> **版本**: v1.0  
> **部门**: 客房部  
> **生效日期**: 2024-01-01

---

## 1. 目的

确保客房清洁质量统一、高效，为宾客提供干净、舒适、安全的住宿环境。

## 2. 适用范围

适用于所有客房服务员进行退房清洁(Vacant Clean)和住客房清洁(Stayover)。

## 3. 所需工具/材料

### 3.1 清洁工具
- 工作车（布草车）
- 吸尘器
- 抹布（分色管理：蓝色-卫生间，黄色-地面，绿色-家具，白色-杯具）
- 拖把
- 扫帚簸箕
- 刮水器
- 刷子

### 3.2 清洁剂
- 多功能清洁剂
- 玻璃清洁剂
- 浴室清洁剂
- 消毒液
- 地毯去渍剂

### 3.3 布草及用品
- 床单、被套、枕套
- 毛巾（浴巾/面巾/地巾）
- 客用品（洗漱用品/茶包/矿泉水等）

## 4. 操作步骤

### 准备阶段 (2分钟)

**步骤1: 检查工作车**
- 布草补充充足
- 客用品齐全
- 清洁工具完好
- 垃圾袋足够

**步骤2: 确认房态**
- 核对房间号
- 确认房态（VD/VCI/OC）
- 检查DND（请勿打扰）状态

**步骤3: 敲门程序**

标准动作：
1. 按门铃或轻敲3下
2. 报身份："客房服务，Housekeeping"
3. 等待3秒
4. 再次敲门
5. 无回应后用钥匙开门
6. 进入后再报："客房服务"

### 退房清洁流程 (30-40分钟)

#### 步骤1: 开窗通风 (1分钟)
- 打开窗户/空调通风
- 拉开窗帘

#### 步骤2: 收垃圾 (3分钟)
- 收集所有垃圾
- 清空烟灰缸
- 更换垃圾袋
- 注意检查抽屉/床头柜

#### 步骤3: 撤布草 (5分钟)

**撤床程序**:
1. 撤枕套（检查枕芯）
2. 撤被套（检查被芯）
3. 撤床单（检查床垫）

**撤卫生间布草**:
1. 撤浴巾、面巾
2. 撤地巾、方巾
3. 放入布草袋

**检查**: 清点迷你吧消费，报告前台

#### 步骤4: 铺床 (8分钟)

**16步铺床法**:

1. 将床拉离床头板
2. 检查床垫（翻转/调头）
3. 铺第一条床单（包角）
4. 铺第二条床单（中线对齐）
5. 铺被套（被套开口远离房门）
6. 套被芯（四角对齐）
7. 整理被子（中线对齐）
8. 床头回折30cm
9. 床尾包角
10. 塞入床垫下
11. 铺枕套（开口远离床头）
12. 整理枕头（饱满挺括）
13. 放回床头
14. 整理床裙（如有）
15. 推回床头板
16. 整体整理

**质量标准**:
- 三线合一（床单中线、被套中线、枕头中线）
- 四角饱满
- 表面平整无褶皱
- 床头回折一致

#### 步骤5: 卫生间清洁 (10分钟)

**清洁顺序**: 从上到下，从干到湿

1. **镜面**: 玻璃清洁剂，无水渍
2. **洗手台**: 清洁剂擦拭，龙头抛光
3. **马桶**: 清洁剂刷洗，消毒，座圈重点清洁
4. **浴缸/淋浴**: 清洁剂刷洗，玻璃门除垢
5. **墙面**: 擦拭瓷砖，除霉点
6. **地面**: 清洁，消毒，擦干
7. **地漏**: 清理毛发，倒消毒液

**补充用品**:
- 洗漱用品（牙刷/牙膏/梳子/浴帽/香皂/洗发水/沐浴露）
- 毛巾（按标准摆放）
- 卫生纸（折叠三角）
- 地巾

#### 步骤6: 抹尘 (5分钟)

**顺序**: 从上到下，从里到外，顺时针

重点部位：
- 家具表面
- 电器（电视/遥控器/电话）
- 灯具
- 窗户/窗台
- 门/门框
- 踢脚线

**标准**: 无灰尘、无水渍、无指印

#### 步骤7: 补充客用品 (3分钟)

**检查清单**:
- [ ] 矿泉水（2瓶）
- [ ] 茶包/咖啡
- [ ] 杯具（清洁）
- [ ] 文具
- [ ] 便签纸
- [ ] 洗衣袋/鞋袋
- [ ] 衣架（数量完整）

#### 步骤8: 吸尘 (3分钟)

**顺序**: 从里到外

重点：
- 地毯全面吸尘
- 家具下方
- 墙角线
- 窗帘除尘

#### 步骤9: 最后检查 (2分钟)

**自查清单**:
- [ ] 床铺整齐
- [ ] 卫生间无水渍
- [ ] 家具复位
- [ ] 灯具关闭
- [ ] 空调适宜（24-26°C）
- [ ] 窗帘整齐
- [ ] 无遗留清洁工具
- [ ] 无异味

### 住客房清洁 (Stayover) (15-20分钟)

**与退房清洁区别**:
- 不换床单（除非客人要求或3天未换）
- 整理个人物品（不乱动）
- 重点清洁卫生间
- 补充客用品
- 简单整理床铺

## 5. 质量标准

### 5.1 72项检查点（核心）

**卧室** (25项):
- 床铺平整、无毛发
- 枕头饱满、无异味
- 家具无灰尘
- 地面无垃圾
- 窗户清洁
- 空调滤网清洁
- 电视/遥控正常
- 电话清洁
- 迷你吧齐全
- 杯具清洁

**卫生间** (20项):
- 镜面无水渍
- 龙头光亮
- 台面无水渍
- 马桶清洁消毒
- 浴缸/淋浴无皂垢
- 地面干燥清洁
- 毛巾整齐
- 用品齐全
- 无异味
- 地漏清洁

### 5.2 时效标准

| 房型 | 退房清洁 | 住客房清洁 |
|------|----------|------------|
| 标准间 | 30分钟 | 15分钟 |
| 套房 | 50分钟 | 25分钟 |

## 6. 常见问题

**Q: 发现客房物品损坏怎么办？**
A: 立即报告领班，拍照记录，不要擅自处理。

**Q: 发现客人私人物品怎么办？**
A: 整理归位，不乱动，如贵重物品立即报告。

**Q: 客人房间挂着DND牌但已超时？**
A: 报告主管，由主管决定如何处理，不要擅自进入。

## 7. 安全注意事项

1. **开门程序**: 必须按标准敲门程序
2. **房门敞开**: 清洁时房门保持敞开
3. **工具管理**: 清洁车不挡通道
4. **化学品**: 妥善保管，不混用
5. **个人防护**: 必要时戴手套

---

**培训要求**: 
- 新员工需通过铺床实操考核（16步标准）
- 清洁质量需通过主管查房考核
- 每月技能比武

**考核标准**: 
- 主管查房合格率>95%
- 客人投诉率<1%
