# 现金管理政策 (Cash Handling P&P)

> **文档编号**: FI-PP-001  
> **版本**: v1.0  
> **部门**: 财务部  
> **生效日期**: 2024-01-01

---

## 1. 目的

规范酒店现金管理，确保资金安全，防止差错和舞弊。

## 2. 适用范围

适用于所有涉及现金收付的岗位和场景。

## 3. 现金收取政策

### 3.1 收银备用金

**备用金额度**:
| 岗位 | 备用金额度 |
|------|------------|
| 前台收银 | ¥5,000 |
| 餐厅收银 | ¥3,000 |
| 其他收银点 | ¥1,000 |

**管理要求**:
- 领用需登记
- 每日盘点
- 差额自负

### 3.2 现金收款程序

**收款流程**:
1. 礼貌问候客人
2. 确认金额
3. 当面清点现金
4. 验钞（大额）
5. 开具收据/发票
6. POS机录入
7. 找零（如有）
8. 道谢

**验钞要求**:
- 50元及以上面额必须验钞
- 使用验钞机
- 可疑钞票要求更换

### 3.3 大额现金

**定义**: 单笔收款≥¥5,000

**处理程序**:
1. 报告主管
2. 双人验钞
3. 及时存入保险箱
4. 减少柜台留存

## 4. 现金缴存政策

### 4.1 缴存时限

| 金额 | 缴存时限 |
|------|----------|
| 正常营业款 | 当日 |
| 大额现金(>1万) | 即时 |
| 营业结束 | 全部缴存 |

### 4.2 缴存程序

**封包**:
1. 清点现金
2. 填写缴款单
3. 装入封包袋
4. 封口签名
5. 投入保险箱

**保险箱管理**:
- 密码+钥匙分离管理
- 开启需双人
- 记录存取

## 5. 现金差错处理

### 5.1 长款处理

**定义**: 实收>应收

**处理程序**:
1. 复核账目
2. 查找原因
3. 无法查明：
   - 当日上缴
   - 记入营业外收入
4. 记录备案

### 5.2 短款处理

**定义**: 实收<应收

**处理程序**:
1. 复核账目
2. 查找原因
3. 确认差错：
   - 责任人补足
   - 书面说明
4. 记录备案
5. 累计3次：纪律处分

### 5.3 差错标准

| 差错类型 | 允许范围 | 处理 |
|----------|----------|------|
| 长款 | 任何金额 | 查明/上缴 |
| 短款 | ≤¥50/月 | 责任人承担 |
| 短款 | >¥50 | 责任人承担+书面警告 |

## 6. 现金安全

### 6.1 柜台安全

**要求**:
- 现金不离开收银台
- 抽屉随时上锁
- 视线不离开现金
- 离开岗位现金入柜

### 6.2 防盗防抢

**预防措施**:
- 安装监控
- 安装警报
- 夜间锁门
- 限额留存

**应急处理**:
1. 保护生命安全
2. 记住特征
3. 立即报警
4. 保护现场
5. 报告上级

### 6.3 保险箱管理

**开启权限**:
- 财务经理：主密码
- 出纳：钥匙
- 开启必须双人

**盘点**:
- 每日盘点备用金
- 每周盘点保险箱
- 每月全面盘点

## 7. 备用金管理

### 7.1 领用

**程序**:
1. 填写领用单
2. 部门经理审批
3. 财务领取
4. 签字确认

### 7.2 使用

**范围**:
- 小额采购
- 紧急支出
- 找零备用

**禁止**:
- 私人借用
- 超范围使用
- 白条抵库

### 7.3 报销

**时限**: 发生后7天内
**单据**: 发票/收据+审批单
**程序**: 填写报销单→审批→财务审核→冲账/补款

## 8. 职责分工

| 角色 | 职责 |
|------|------|
| 财务总监 | 现金政策制定、大额审批 |
| 财务经理 | 保险箱管理、现金监盘 |
| 出纳 | 现金收付、银行存取 |
| 收银 | 日常收银、缴存 |

## 9. 审计要求

**内部审计**:
- 月度现金盘点
- 季度突击检查
- 年度全面审计

**审计内容**:
- 账实相符
- 手续完备
- 安全合规
