# 入住SOP (Check-in SOP)

> **文档编号**: FO-SOP-001  
> **版本**: v1.0  
> **部门**: 前厅部  
> **生效日期**: 2024-01-01

---

## 1. 目的

确保宾客入住流程高效、专业、温馨，给客人留下良好的第一印象。

## 2. 适用范围

适用于所有前台接待员办理散客/团队入住。

## 3. 所需工具/材料

- PMS系统登录
- 房卡（已制卡或空白卡）
- 入住登记表（Reg Card）
- 证件识别仪
- POS机（押金收取）
- 欢迎卡/早餐券
- 房卡套（含酒店信息）

## 4. 操作步骤

### 步骤1: 热情问候 (0-30秒)

**标准话术**:
> "您好，欢迎光临[酒店名]，请问有什么可以帮您？"

**动作**:
- 目光接触，微笑
- 站立或坐姿端正
- 如正在接听电话，示意客人稍等

### 步骤2: 确认预订 (30秒-1分钟)

**标准话术**:
> "请问您有预订吗？请问预订姓名是？"

**操作**:
1. 在PMS中查询预订
2. 核对预订信息（房型/天数/价格）
3. 确认特殊要求（高楼层/无烟房等）

**如无预订**:
> "请问您需要什么样的房型？住几晚？"
- 查询实时房态
- 推荐合适房型
- 报价并确认

### 步骤3: 证件登记 (1-2分钟)

**标准话术**:
> "请出示您的身份证件，谢谢。"

**操作**:
1. 接取证件（双手）
2. 使用证件识别仪读取信息
3. 核对证件有效期
4. 人证比对

**PMS录入**:
- 自动填充客人信息
- 补充联系方式
- 确认房价

### 步骤4: 押金收取 (1-2分钟)

**标准话术**:
> "房费已含/不含早餐。需要收取押金[金额]，您方便刷卡还是付现金？"

**操作**:
1. 计算押金金额（房费+杂费预估）
2. POS机预授权/现金收取
3. 打印押金单
4. 请客人签字确认

### 步骤5: 制作房卡 (30秒)

**操作**:
1. 选择对应房间号
2. 制作房卡（写卡/发卡）
3. 测试房卡
4. 装入房卡套

### 步骤6: 信息介绍 (1-2分钟)

**标准话术**:
> "您的房间在[楼层]，房间号是[房号]。早餐在[地点]，时间是[时段]。电梯在您右手边。"

**必说信息**:
- [ ] 房间号+楼层
- [ ] WiFi密码
- [ ] 早餐时间/地点
- [ ] 健身房/泳池开放时间（如有）
- [ ] 紧急联系电话

### 步骤7: 递交物品 (30秒)

**标准动作**:
1. 双手递交房卡
2. 递交早餐券（如适用）
3. 递交发票信息卡（如需开票）

**标准话术**:
> "这是您的房卡和早餐券，祝您入住愉快！"

### 步骤8: 完成登记 (30秒)

**操作**:
1. 入住登记表签字（如需要）
2. 系统完成Check-in
3. 证件归还（双手）
4. 行李服务询问

**行李服务话术**:
> "请问您有行李需要帮您送到房间吗？"

## 5. 质量标准

| 指标 | 标准 |
|------|------|
| 办理时长 | ≤3分钟（有预订）/ ≤5分钟（无预订） |
| 微笑服务 | 100% |
| 目光接触 | 100% |
| 双手递物 | 100% |
| 信息完整录入 | 100% |

## 6. 常见问题

**Q: 客人证件消磁/无法读取怎么办？**
A: 手工录入信息，仔细核对证件照片与本人是否一致。

**Q: 预订房型已满怎么办？**
A: 向客人致歉，免费升级至更高房型，如无法升级则安排同档次附近酒店并承担差价。

**Q: 客人要求提前入住（房间未准备好）？**
A: 询问是否可以等待，提供行李寄存和休息区；如无法接受则提供钟点房或建议附近咖啡厅。

## 7. 流程图

```
[客人到达]
    ↓
[热情问候] ──→ 电话/其他事务中 → [致歉并示意等待]
    ↓
[确认预订] ──→ 无预订 → [查询房态推荐房型]
    ↓
[证件登记]
    ↓
[押金收取]
    ↓
[制作房卡]
    ↓
[信息介绍]
    ↓
[递交物品]
    ↓
[完成登记] ──→ 需行李服务 → [通知行李员]
    ↓
[送别客人]
```
