# 招聘政策 (Recruitment P&P)

> **文档编号**: HR-PP-001  
> **版本**: v1.0  
> **部门**: 人力资源部  
> **生效日期**: 2024-01-01

---

## 1. 目的

规范招聘流程，确保招聘质量，吸引和选拔优秀人才。

## 2. 适用范围

适用于酒店所有岗位的招聘。

## 3. 招聘原则

- **公平竞争**: 机会均等，择优录取
- **德才兼备**: 品德优先，能力并重
- **人岗匹配**: 适才适用，发挥所长
- **合规合法**: 遵守劳动法，禁止歧视

## 4. 招聘流程

### 4.1 需求申请

**申请条件**:
- 编制内有空缺
- 编制外需经GM特批
- 离职补充或新增岗位

**审批流程**:
```
部门申请 → 部门总监审批 → HR审核编制 → GM审批 → 发布招聘
```

### 4.2 招聘渠道

| 渠道 | 适用岗位 | 成本 |
|------|----------|------|
| 内部推荐 | 全岗位 | 低（推荐奖） |
| 招聘网站 | 基层/专业 | 中 |
| 猎头 | 高管/稀缺 | 高 |
| 校园招聘 | 管培生/储备 | 中 |
| 劳务派遣 | 基层/临时 | 低 |

### 4.3 面试流程

| 岗位级别 | 面试轮次 | 面试官 |
|----------|----------|--------|
| 基层员工 | 2轮 | HR+部门经理 |
| 主管 | 3轮 | HR+部门总监+GM |
| 经理及以上 | 3-4轮 | HR+相关部门+GM+业主 |

### 4.4 背景调查

**必查岗位**:
- 财务相关
- 安保相关
- 高管
- 涉及现金岗位

**调查内容**:
- 学历验证
- 工作经历验证
- 犯罪记录（必要时）
- 信用记录（财务岗）

## 5. 录用标准

### 5.1 基本条件

**年龄**:
- 满18周岁
- 退休返聘需特批

**健康**:
- 持有效健康证（餐饮/客房）
- 无传染性疾病
- 无职业禁忌症

**证件**:
- 身份证真实有效
- 相关资格证书（如有）

### 5.2 禁止录用情形

- 未满18周岁
- 伪造证件/简历
- 有犯罪记录（安保/财务岗）
- 与在职员工有亲属关系（特殊岗位）
- 竞业限制期内

## 6. 入职程序

### 6.1 入职准备

**材料清单**:
- [ ] 身份证复印件
- [ ] 学历证明
- [ ] 健康证（如需）
- [ ] 离职证明
- [ ] 照片（1寸/2寸）
- [ ] 银行卡
- [ ] 资格证书

### 6.2 入职流程

**第一天**:
1. 签订劳动合同
2. 领取员工手册
3. 办理工牌/工服
4. 录入考勤系统
5. 安排宿舍（如需）
6. 部门报到

**第一周**:
- 入职培训（企业文化/规章制度）
- 部门培训（岗位知识）
- 师傅带教

**第一个月**:
- 试用期考核
- 转正评估

## 7. 试用期管理

### 7.1 试用期期限

| 合同期限 | 试用期 |
|----------|--------|
| 3个月-1年 | 1个月 |
| 1-3年 | 2个月 |
| 3年以上/无固定 | 6个月 |

### 7.2 试用期考核

**考核节点**:
- 周度：师傅反馈
- 月度：书面评估
- 期满：转正评估

**考核内容**:
- 工作态度
- 工作能力
- 学习能力
- 团队协作
- 遵守制度

**转正条件**:
- 考核合格（>70分）
- 无重大违纪
- 健康证有效

## 8. 内部招聘

### 8.1 内部晋升

**原则**:
- 优先内部选拔
- 公开竞聘
- 择优晋升

**程序**:
1. 发布内部招聘信息
2. 员工申请
3. 部门推荐
4. 面试评估
5. 公示
6. 办理调动

### 8.2 轮岗制度

**适用对象**:
- 管培生
- 储备干部
- 高潜员工

**轮岗周期**: 3-6个月/岗

## 9. 招聘纪律

### 9.1 面试官纪律

- 公平公正，不徇私情
- 保密面试信息
- 及时反馈结果
- 不得收受礼品

### 9.2 违规处理

| 违规行为 | 处罚 |
|----------|------|
| 徇私录用 | 解除劳动合同 |
| 泄露考题 | 警告/降职 |
| 歧视性招聘 | 纪律处分 |
