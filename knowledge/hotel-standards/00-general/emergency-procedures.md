# 紧急程序 (Emergency Procedures)

> **版本**: v1.0

## 1. 目的

建立酒店紧急情况响应机制，确保在突发事件中能够快速、有序地采取行动，保护生命安全，减少财产损失。

## 3. 火灾应急

### 3.1 发现火情
1. **大声呼叫**："着火了！"
2. **按响警报**：按下最近的手动报警按钮
3. **报告位置**：拨打内部电话报告火情位置和大小
4. **初期灭火**：如火势小，使用灭火器扑救
5. **疏散**：如火势大，立即疏散

### 3.2 疏散程序
1. **听从指挥**：听从广播和工作人员指挥
2. **走楼梯**：禁止使用电梯
3. **低姿前进**：烟雾中弯腰或匍匐前进
4. **湿布捂口鼻**：如有条件，用湿毛巾捂住口鼻
5. **到集合点**：前往指定集合点
6. **清点人数**：协助清点人员

### 3.3 灭火器使用
**提**起灭火器 → **拔**出保险销 → **握**住喷管 → **压**下压把 → 对准火焰**根部**喷射

## 4. 医疗急救

### 4.1 发现伤病
1. 评估现场安全
2. 判断意识和呼吸
3. 呼叫帮助（拨打120）
4. 实施急救
5. 等待专业救援

### 4.2 心肺复苏 (CPR)
1. 确认无呼吸/无心跳
2. 呼叫120并开始CPR
3. 胸外按压：每分钟100-120次，深度5-6cm
4. 人工呼吸：30次按压后2次人工呼吸
5. 持续至专业救援到达

### 4.3 AED使用
1. 打开AED电源
2. 按语音提示贴电极片
3. 分析心律（不要接触患者）
4. 如需电击，确保无人接触患者后按下电击按钮
5. 继续CPR

### 4.4 出血处理
1. 直接压迫伤口
2. 抬高伤肢
3. 如严重，使用止血带（记录时间）
4. 尽快送医

### 4.5 烧伤处理
1. 冷却烧伤部位（流动冷水15-20分钟）
2. 覆盖干净敷料
3. 不要涂抹任何药物
4. 严重烧伤立即送医

## 5. 自然灾害

### 5.1 地震
**震时**：
- 就地躲避在坚固家具旁
- 远离窗户、玻璃、高处物品
- 如在室外，远离建筑物
- 不要乘电梯

**震后**：
- 检查伤情
- 检查火源、电源、气源
- 有序疏散到开阔地带
- 听从指挥，不要慌乱

### 5.2 台风/暴雨
**准备**：
- 关注天气预报
- 加固门窗
- 准备应急物资
- 检查排水系统

**应对**：
- 留在室内安全处
- 远离窗户
- 储备饮用水
- 储备食物

## 6. 治安事件

### 6.1 盗窃发现
1. 保护现场，不要触碰
2. 立即报告安保部
3. 配合调查
4. 清点损失
5. 报警（如需要）

### 6.2 暴力事件
1. 确保自身安全
2. 立即报警（110）
3. 报告安保部
4. 协助疏散周围人员
5. 记住嫌疑人特征
6. 保护现场

### 6.3 炸弹威胁
1. 保持冷静，记录详细信息
2. 立即报告安保部和总值班
3. 不要触碰可疑物品
4. 听从疏散指令
5. 协助疏散

## 7. 设备故障

### 7.1 停电
1. 保持冷静，安抚客人
2. 启动应急照明
3. 启动发电机
4. 报告工程部
5. 关注电梯困人情况

### 7.2 电梯困人
1. 安抚被困人员
2. 通知工程部
3. 联系电梯维保
4. 必要时拨打119
5. 不要强行撬门

### 7.3 燃气泄漏
1. 立即关闭燃气总阀
2. 开窗通风
3. 禁止明火和电器开关
4. 疏散人员
5. 报告工程部和燃气公司

### 7.4 水管爆裂
1. 关闭水阀
2. 报告工程部
3. 转移重要物资
4. 排水防滑
5. 设置警示标志

## 8. 食品安全事故

### 8.1 食物中毒
1. 立即停止供应可疑食品
2. 保留食物样本
3. 协助患者就医
4. 报告卫生部门
5. 配合调查

## 9. 客人特殊情况

### 9.1 客人在房间意外
1. 立即到现场
2. 评估情况
3. 呼叫急救（如需要）
4. 保护现场
5. 报告上级
6. 联系家属（如需要）

### 9.2 客人失踪
1. 确认最后出现时间和地点
2. 查看监控
3. 检查房间
4. 询问同行人员
5. 报告安保部
6. 必要时报警

## 10. 媒体应对

### 10.1 发生紧急事件时
1. 员工不得擅自接受媒体采访
2. 指引媒体联系酒店公关部或总值班
3. 统一由指定发言人对外发布信息
4. 不传播未经证实的信息

## 11. 应急物资

### 11.1 配置位置
- 急救箱：前台、厨房、工程部、安保部
- 灭火器：各楼层、公共区域
- 应急灯：各楼层、疏散通道
- A.OE.D：前台、健身房（如有）
- 防汛物资：地下室入口

### 11.2 定期检查
- 每月检查应急物资
- 及时补充更换
- 记录检查情况

## 12. 培训与演练

### 12.1 培训要求
- 新员工入职培训必须包含应急程序
- 每年复训
- 重点岗位专项培训

### 12.2 演练计划
- 消防疏散演练：每半年一次
- 急救培训：每年一次
- 综合演练：每年一次

## 13. 事后处理

### 13.1 事件报告
所有紧急事件必须填写《紧急事件报告表》，包括：
- 事件发生时间、地点
- 事件经过
- 采取的措施
- 损失情况
- 改进建议

### 13.2 调查分析
- 成立调查组
- 分析原因
- 制定改进措施
- 更新应急预案

### 13.3 恢复运营
- 确保安全后恢复
- 检查设施设备
- 心理辅导（如需要）
- 总结报告
