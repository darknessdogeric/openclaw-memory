# 食品安全政策 (Food Safety P&P)

> **文档编号**: FB-PP-001  
> **版本**: v1.0  
> **部门**: 餐饮部  
> **生效日期**: 2024-01-01

---

## 1. 目的

确保酒店食品安全，预防食物中毒，保障宾客和员工健康，符合法律法规要求。

## 2. 适用范围

适用于酒店所有餐饮区域：
- 全日制餐厅
- 中餐厅
- 西餐厅
- 宴会厨房
- 饼房
- 酒吧
- 客房送餐
- 员工餐厅

## 3. 政策内容

### 3.1 HACCP体系

**7个关键控制点 (CCP)**:

| CCP | 控制点 | 关键限值 | 监控方法 |
|-----|--------|----------|----------|
| CCP1 | 采购验收 | 合格供应商、新鲜度 | 索证索票、感官检查 |
| CCP2 | 冷藏储存 | 0-4°C | 温度记录 |
| CCP3 | 冷冻储存 | <-18°C | 温度记录 |
| CCP4 | 烹饪加热 | 中心温度>70°C | 温度计检测 |
| CCP5 | 保温/冷却 | 热食>60°C，冷食<8°C | 温度记录 |
| CCP6 | 交叉污染 | 生熟分开 | 目视检查 |
| CCP7 | 个人卫生 | 健康证、清洁操作 | 每日检查 |

### 3.2 食品储存政策

**先进先出 (FIFO)**:
- 所有食品必须标注进货日期和开封日期
- 先使用旧批次
- 定期检查保质期

**储存温度**:
| 类别 | 温度 | 时限 |
|------|------|------|
| 冷藏生鲜 | 0-4°C | 1-2天 |
| 冷藏熟食 | 0-4°C | 2-3天 |
| 冷冻食品 | <-18°C | 按包装标示 |
| 干货 | 常温干燥 | 按包装标示 |

**储存原则**:
- 生熟分开
- 上干下湿
- 离地15cm
- 离墙10cm
- 分类存放

### 3.3 个人卫生政策

**健康证**:
- 所有餐饮从业人员必须持有效健康证
- 健康证每年更新
- 新入职员工须持健康证上岗

**个人卫生要求**:
| 项目 | 要求 |
|------|------|
| 指甲 | 短而干净，不得涂指甲油 |
| 头发 | 戴帽子/发网，不得外露 |
| 首饰 | 不得佩戴（婚戒除外） |
| 手部 | 勤洗手，必要时戴手套 |
| 工作服 | 清洁、整洁、专岗专用 |
| 伤口 | 必须包扎并戴双层手套 |

**洗手程序**:
1. 湿手
2. 涂洗手液
3. 搓洗20秒（手心/手背/指缝/拇指/指尖/手腕）
4. 冲洗
5. 擦干
6. 消毒

**必须洗手的情况**:
- 上岗前
- 如厕后
- 处理垃圾后
- 接触生食后
- 打喷嚏/咳嗽后
- 用餐后
- 抽烟后

### 3.4 清洁消毒政策

**餐具消毒**:
- 物理消毒：洗碗机>82°C
- 化学消毒：含氯消毒剂>200ppm，浸泡5分钟
- 保洁：消毒后存放在保洁柜

**工器具消毒**:
- 生熟用具分开
- 使用后清洗消毒
- 定位存放

**环境消毒**:
- 每日营业后清洁消毒
- 每周深度清洁
- 每月四害防治

### 3.5 食品留样政策

**留样要求**:
- 所有出品食品必须留样
- 留样量：>125g
- 留样时间：48小时
- 留样条件：0-4°C冷藏
- 留样记录：品名、时间、制作人

**留样容器**:
- 专用留样盒
- 加盖密封
- 标签完整

### 3.6 应急预案

**食物中毒应急**:
1. 立即报告值班经理和厨师长
2. 停止可疑食品供应
3. 封存留样
4. 协助就医
5. 报告卫生部门
6. 配合调查

## 4. 职责分工

| 角色 | 职责 |
|------|------|
| 餐饮总监 | 食品安全第一责任人 |
| 厨师长 | 厨房食品安全管理 |
| 管事部经理 | 清洁消毒管理 |
| 食品卫生员 | 日常检查、记录、培训 |
| 所有员工 | 执行食品安全规定 |

## 5. 培训要求

**入职培训**:
- 食品安全基础知识（4小时）
- HACCP体系介绍
- 个人卫生要求
- 应急预案

**在职培训**:
- 每月食品安全例会
- 每季度专项培训
- 新员工带教

## 6. 检查与记录

**日常检查**:
- 厨房自查（每日）
- 管事部检查（每日）
- 餐饮部抽查（每周）

**记录保存**:
- 温度记录（2年）
- 留样记录（2年）
- 检查记录（2年）
- 培训记录（员工在职期间+2年）
