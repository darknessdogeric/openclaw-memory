# 厨房操作SOP (Kitchen SOP)

> **文档编号**: FB-SOP-001  
> **版本**: v1.0  
> **部门**: 餐饮部  
> **生效日期**: 2024-01-01

---

## 1. 目的

规范厨房操作流程，确保食品安全、出品质量和操作安全。

## 2. 适用范围

适用于酒店所有厨房区域。

## 3. 开档准备 (Opening)

### 3.1 时间
**每日**: 根据营业时间提前2小时

### 3.2 准备工作

**个人卫生**:
- [ ] 穿戴清洁工作服
- [ ] 洗手消毒
- [ ] 戴帽子/发网
- [ ] 检查指甲

**设备检查**:
- [ ] 炉灶点火正常
- [ ] 烤箱预热
- [ ] 蒸箱加水
- [ ] 冰箱温度正常
- [ ] 排风开启

**原料准备**:
- [ ] 解冻当日所需原料
- [ ] 检查原料新鲜度
- [ ] 预处理和初加工
- [ ] 准备调味料

**工具准备**:
- [ ] 刀具消毒
- [ ] 砧板清洁
- [ ] 盛具准备
- [ ] 检查燃气/电路

## 4. 原料接收与储存

### 4.1 验收标准

**生鲜原料**:
- 肉类：色泽正常、无异味、检疫合格
- 海鲜：鲜活或冰鲜、无异味
- 蔬菜：新鲜、无黄叶、无虫害

**冷冻原料**:
- 包装完整
- 无解冻再冻痕迹
- 在保质期内

**干货**:
- 包装完好
- 无霉变/虫蛀
- 在保质期内

### 4.2 验收程序

1. 核对送货单与订货单
2. 检查数量
3. 质量验收
4. 索证索票（检疫证/检验报告）
5. 合格签字入库
6. 不合格退货

### 4.3 储存要求

**分类存放**:
```
冷藏冰箱：
├── 上层：熟食/即食食品
├── 中层：半成品
└── 下层：生鲜原料

冷冻冰箱：
├── 肉类
├── 海鲜
└── 其他

干货库：
├── 粮油
├── 调料
└── 干货
```

**标识管理**:
- 品名
- 进货日期
- 开封日期
- 保质期
- 责任人

## 5. 食品加工SOP

### 5.1 粗加工

**蔬菜加工**:
1. 择捡（去除黄叶/烂叶）
2. 清洗（流动水冲洗）
3. 浸泡（必要时，去除农药）
4. 切配（按标准规格）
5. 存放（加盖，标识）

**肉类加工**:
1. 解冻（冷藏解冻或流水解冻）
2. 清洗
3. 分割
4. 腌制（按配方）
5. 存放（生熟分开）

### 5.2 切配标准

| 原料 | 规格 | 用途 |
|------|------|------|
| 土豆丝 | 3mm×3mm，均匀 | 炒/醋溜 |
| 肉片 | 3cm×2cm×0.3cm | 炒/水煮 |
| 鱼片 | 0.5cm厚 | 水煮/酸菜 |
| 丁 | 1.5cm×1.5cm | 宫保/炒 |
| 块 | 3cm×3cm | 红烧/炖 |

### 5.3 烹饪标准

**炒菜**:
1. 热锅凉油
2. 爆香（葱姜蒜）
3. 下主料
4. 调味
5. 翻炒成熟
6. 勾芡（如需要）
7. 出锅装盘

**油炸**:
- 油温控制：
  - 低油温（3-4成，90-120°C）：滑油
  - 中油温（5-6成，150-180°C）：炸制
  - 高油温（7-8成，180-210°C）：复炸

**蒸制**:
- 水开后上锅
- 控制时间（防止过火）
- 蒸汽充足

**烧烤/烤箱**:
- 预热至指定温度
- 定时烘烤
- 翻面（如需要）

### 5.4 出品标准

**温度标准**:
- 热食：>60°C（上桌时）
- 冷食：<8°C

**时间标准**:
- 点菜后15分钟内出品（特殊情况除外）
- 高峰时段20分钟

**质量标准**:
- 色：色泽自然、诱人
- 香：香气浓郁、无异味
- 味：咸淡适中、口味纯正
- 形：造型美观、摆盘规范
- 器：餐具清洁、温度适宜

## 6. 收档程序 (Closing)

### 6.1 时间
营业结束后1小时内完成

### 6.2 收档内容

**原料处理**:
- 剩余原料分类存放
- 易腐原料优先使用
- 变质原料废弃

**清洁工作**:
- 清洁操作台
- 清洗锅具/器具
- 清洁炉灶/烤箱
- 拖地
- 垃圾清理

**设备关闭**:
- 关闭燃气阀门
- 关闭电器设备（除冰箱外）
- 关闭排风

**安全检查**:
- 检查水阀
- 检查燃气
- 检查电源
- 锁门

## 7. 五常法管理

**常整理**:
- 区分要与不要
- 不要的清理

**常整顿**:
- 定位摆放
- 标识清楚

**常清扫**:
- 每日清洁
- 每周大扫除

**常清洁**:
- 维持整洁
- 形成习惯

**常素养**:
- 遵守制度
- 养成习惯

## 8. 安全操作

### 8.1 消防安全
- 油锅起火：盖锅盖，关燃气，不要用水
- 燃气泄漏：关阀、通风、禁火、报修
- 灭火器位置清楚

### 8.2 操作安全
- 刀具使用：注意力集中，刀刃向内
- 防滑：地面保持干燥
- 防烫：使用防烫手套/工具
- 机械安全：按规程使用设备

### 8.3 食品安全
- 生熟分开
- 烧熟煮透
- 剩菜处理：充分加热或废弃

- 新员工必须通过食品安全和岗位技能培训
- 每季度技能考核

**考核标准**:
- 出品合格率>98%
- 食品安全零事故
- 操作规范执行率100%
