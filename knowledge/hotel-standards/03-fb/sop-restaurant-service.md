# 餐厅服务SOP (Restaurant Service SOP)

> **文档编号**: FB-SOP-003  
> **版本**: v1.0  
> **部门**: 餐饮部-餐厅  
> **生效日期**: 2024-01-01

---

## 1. 目的

规范餐厅服务流程，确保为客人提供专业、周到、优雅的用餐体验。

## 2. 适用范围

适用于酒店全日制餐厅、中餐厅、西餐厅、特色餐厅的服务工作。

## 3. 餐前准备

### 3.1 个人准备
- 仪容仪表检查
- 制服整洁
- 口气清新
- 熟悉当日特色/促销

### 3.2 餐厅准备（每日营业前1小时）

**环境**:
- [ ] 灯光调节
- [ ] 空调温度（22-24°C）
- [ ] 背景音乐
- [ ] 开窗通风

**餐桌**:
- [ ] 桌椅对齐
- [ ] 台布平整
- [ ] 餐具摆放标准
- [ ] 口布折花
- [ ] 调味品齐全

**用品**:
- [ ] 菜单检查（数量/清洁）
- [ ] 点菜单/酒水单
- [ ] 托盘/服务车
- [ ] 开瓶器
- [ ] 冰块充足

**检查**:
- [ ] 设备正常（POS/灯光/空调）
- [ ] 卫生间检查
- [ ] 领位台准备

## 4. 迎宾领位

### 4.1 迎接

**标准站位**: 餐厅入口，面向客人来的方向

**迎接话术**:
> "您好，欢迎光临[餐厅名]，请问有预订吗？"

**确认预订**:
- 有预订："请问预订姓名是？" → 确认后引领
- 无预订："请问几位用餐？" → 查看空位 → 引领

### 4.2 领位

**行走**:
- 走在客人左前方1-1.5米
- 步速适中
- 手势引导

**安排座位**:
- 询问偏好（靠窗/安静/吸烟区）
- 根据人数安排合适桌子
- 先为女士/长者拉椅

**拉椅标准**:
- 右脚后撤半步
- 双手扶椅背
- 客人入座时轻推

**递菜单**:
- 打开第一页
- 从客人右侧递上
- "这是菜单，请慢慢看，我稍后为您介绍今日特色。"

## 5. 点菜服务

### 5.1 点菜时机

- 客人入座后3-5分钟
- 或客人示意时
- 不要催促

### 5.2 点菜流程

**自我介绍**:
> "您好，我是[姓名]，今天由我为您服务。请问需要我为您介绍今日特色吗？"

**推荐**:
- 当日特色菜
- 厨师推荐
- 时令菜品
- 配餐建议

**记录**:
- 使用点菜单
- 复述确认
- 询问特殊要求（过敏/忌口）
- 询问上菜顺序

**确认**:
> "您点了[复述菜品]，请问还有什么需要吗？酒水需要现在点吗？"

### 5.3 酒水服务

**推荐**:
> "我们有[红白葡萄酒/啤酒/特色饮品]，需要我为您推荐搭配吗？"

**示瓶**:
- 展示酒瓶标签
- 客人确认后开瓶

**开瓶**:
- 使用酒刀
- 不发出声响
- 检查木塞

**试酒**:
- 主人杯中倒少量
- 请主人品尝确认
- 确认后按顺序倒酒（女士优先）

**倒酒量**:
- 红酒：1/3杯
- 白酒：2/3杯
- 啤酒：8分满

## 6. 上菜服务

### 6.1 上菜顺序

1. 面包/黄油
2. 头盘/冷菜
3. 汤
4. 主菜
5. 配菜
6. 甜点
7. 咖啡/茶

### 6.2 上菜标准

**托盘使用**:
- 左手托盘，五指分开
- 重心靠近身体
- 平稳行走

**上菜位置**:
- 从客人右侧上菜（中餐可灵活）
- 不越过客人头顶
- 先女后男，先宾后主

**报菜名**:
> "这是您点的[菜名]，请慢用。"

**摆盘**:
- 转盘顺时针
- 主菜正对主人
- 盘子距桌边2cm

**分菜服务**（如需）:
- 使用公勺公筷
- 均匀分配
- 先女士后男士

## 7. 巡台服务

### 7.1 巡台频率
- 每10-15分钟巡台一次
- 观察客人需求
- 及时响应

### 7.2 服务内容

**更换骨碟**:
- 有残渣时
- 使用托盘
- 从右侧更换

**添茶水/酒水**:
- 杯中少于1/3时添加
- 询问"是否需要添加[饮品]？"

**整理桌面**:
- 清理空盘
- 调整餐具位置
- 保持整洁

**响应需求**:
- 及时回应客人召唤
- 使用手势回应（远距离）

## 8. 结账服务

### 8.1 确认结账

**询问**:
> "请问需要现在结账吗？"

**确认账单**:
- 核对桌号
- 核对菜品
- 核对酒水
- 确认有无折扣

### 8.2 递送账单

**账单夹**:
- 账单正面朝向客人
- 夹子右侧递给主人
- "这是您的账单，请核对。"

### 8.3 收款

**现金**:
- 当面点清
- 复述金额
- "收您[金额]，请稍等。"
- 找零+发票（双手递上）

**刷卡/移动支付**:
- 确认金额
- 引导操作
- 确认支付成功
- 递送小票

**挂账**:
- 确认挂账权限
- 签字确认
- 核对签名

### 8.4 送客

**协助**:
- 拉椅
- 协助取外套
- 提醒携带物品

**道别**:
> "感谢您的光临，期待再次为您服务，请慢走！"

**15度鞠躬**（部分高星级）

## 9. 收台服务

### 9.1 收台顺序
- 先收玻璃器皿
- 再收瓷器
- 最后收金属器皿
- 垃圾单独收

### 9.2 重新摆台
- 按标准快速摆台
- 准备接待下一批客人

## 10. 服务禁忌

- 手指接触餐具食物接触部位
- 说话口水飞溅
- 在服务台整理仪容
- 当众打哈欠/打喷嚏
- 与客人争辩
- 催促客人
- 偷听客人谈话

## 11. 质量标准

| 指标 | 标准 |
|------|------|
| 迎宾响应 | <30秒 |
| 点菜响应 | 3-5分钟 |
| 上菜速度 | 头盘15分钟/主菜20分钟 |
| 巡台频率 | 10-15分钟 |
| 结账响应 | <5分钟 |
| 客人满意度 | >95% |

---

**培训要求**: 餐厅服务专项培训，包括中餐/西餐服务技能、酒水知识、礼仪  
**考核标准**: 服务流程熟练度+客人满意度+销售能力
