# 会议/宴会销售SOP (MICE Sales SOP)

> **文档编号**: SM-SOP-003  
> **版本**: v1.0  
> **部门**: 市场销售部  
> **生效日期**: 2024-01-01

---

## 1. 目的

规范会议、宴会销售全流程，确保客户需求准确传达，活动顺利执行，提升客户满意度和会议收入。

## 2. 适用范围

适用于所有会议、宴会、婚宴、团队用餐的销售与执行。

## 3. 会议类型

| 类型 | 规模 | 特点 | 利润率 |
|------|------|------|--------|
| 商务会议 | 20-200人 | 周期性、稳定性高 | 高 |
| 培训会议 | 50-500人 | 对视听设备要求高 | 中高 |
| 产品发布 | 100-1000人 | 仪式感强、布置要求高 | 高 |
| 婚宴 | 100-500人 | 情感因素重、细节多 | 高 |
| 团队用餐 | 30-300人 | 批量大、标准化 | 中 |

## 4. 销售流程

### 4.1 需求收集（首次接触）

**基本信息**:
- [ ] 活动类型（会议/婚宴/宴会）
- [ ] 预计日期（首选+备选）
- [ ] 预计人数
- [ ] 预算范围
- [ ] 活动时长

**详细需求**:
- [ ] 场地要求（厅高/无柱/自然光等）
- [ ] 座位形式（剧院式/课桌式/宴会式）
- [ ] 餐饮需求（中式/西式/自助）
- [ ] 住宿需求（房间数/房型）
- [ ] 视听设备（LED/投影/音响）
- [ ] 特殊需求（舞台/背景板/鲜花）

**决策信息**:
- [ ] 决策人/关键人
- [ ] 决策时间
- [ ] 竞品情况
- [ ] 历史合作情况

### 4.2 场地带看

**带看前准备**:
- 确认场地可用性
- 按客户需求布置示意
- 准备场地平面图
- 准备样品（菜单/桌布/餐具）

**带看流程（30-45分钟）**:
1. **开场**（5分钟）：了解详细需求变化
2. **场地参观**（15-20分钟）：
   - 宴会厅（主场地）
   - 贵宾休息室
   - 餐饮区域
   - 客房（如需要）
   - 公共区域
3. **功能介绍**（10分钟）：设施设备、服务能力
4. **方案初步沟通**（5分钟）：根据反馈调整
5. **意向确认**（5分钟）：是否进入报价阶段

### 4.3 方案制作（带看后24小时内）

**方案内容**:

**封面**:
- 客户公司名称
- 活动主题
- 提案日期
- 销售经理联系方式

**正文**:
1. **活动概述**
   - 活动类型/日期/人数
   - 场地推荐及理由
   
2. **场地布置方案**
   - 平面图
   - 座位示意图
   - 舞台/背景设计（效果图）
   - 鲜花绿植布置

3. **餐饮方案**
   - 菜单选项（A/B/C套）
   - 酒水方案
   - 服务形式
   - 特殊饮食安排（素食/清真等）

4. **住宿方案**（如需要）
   - 房型分配
   - 价格优惠

5. **视听设备方案**
   - 设备清单
   - 技术人员配置

6. **服务团队**
   - 专属销售经理
   - 宴会服务经理
   - 宴会管家

7. **报价明细**
   - 分项报价
   - 优惠方案
   - 付款条款
   - 取消政策

8. **成功案例**
   - 同类型活动照片
   - 客户评价

9. **酒店简介**
   - 资质/荣誉

### 4.4 方案汇报

**汇报准备**:
- PPT演示
- 场地视频/图片
- 样品（菜单/材质）
- 合同草案

**汇报要点**:
- 强调与需求的匹配度
- 突出差异化优势
- 展示成功案例
- 灵活应对价格谈判

### 4.5 签约

**合同条款**:
- 活动日期/时间
- 场地/餐饮/住宿明细
- 价格及付款方式
- 取消/改期政策
- 双方责任
- 保密条款

**定金**:
- 通常收取总金额的30-50%
- 签约后3-5个工作日内支付

## 5. 活动执行

### 5.1 BEO制作（活动前7-14天）

**BEO（Banquet Event Order）内容**:

| 项目 | 内容 |
|------|------|
| 基本信息 | 客户名称/活动名称/日期/时间 |
| 场地 | 宴会厅/休息室/具体布置 |
| 餐饮 | 菜单/酒水/服务形式/特殊要求 |
| 住宿 | 房间数/房型/入住名单 |
| 视听 | 设备清单/技术需求 |
| 布置 | 舞台/背景/鲜花/桌布颜色 |
| 人员 | 服务人数/特殊安排 |
| 时间表 | 详细时间节点 |
| 联系人 | 客户负责人/酒店负责人 |

**BEO会签**:
- 餐饮部
- 厨房
- 工程部
- 安保部
- 前厅部
- 财务部（挂账审批）

### 5.2 内部协调会（活动前3-7天）

**参会部门**: 餐饮、厨房、工程、安保、前厅、销售

**会议内容**:
- 确认最终人数
- 确认菜单/特殊要求
- 确认设备布置
- 确认停车/动线
- 应急预案
- 分工明确

### 5.3 活动前检查（活动前1天）

**销售经理检查**:
- [ ] 场地清洁
- [ ] 桌椅摆放
- [ ] 舞台搭建
- [ ] 背景板安装
- [ ] 音响测试
- [ ] 灯光测试
- [ ] 空调温度
- [ ] 指示标牌

**与客户确认**:
- 最终人数
- 最终菜单
- 特殊要求
- 到场时间

### 5.4 活动当天执行

**提前到场时间**:
- 销售经理：活动前2小时
- 宴会经理：活动前3小时
- 服务团队：活动前2小时

**执行 checklist**:

**活动前2小时**:
- [ ] 最终检查场地
- [ ] 确认设备正常
- [ ] 确认餐饮准备
- [ ] 确认服务人员到位

**活动前1小时**:
- [ ] 迎接客户负责人
- [ ] 确认流程
- [ ] 处理临时需求

**活动中**:
- [ ] 全程在场协调
- [ ] 监控服务质量
- [ ] 及时处理问题
- [ ] 与客户保持沟通

**活动结束**:
- [ ] 协助结账
- [ ] 收集反馈
- [ ] 送别客户
- [ ] 检查遗留物品

### 5.5 活动后跟进（24-48小时内）

- 发送感谢函
- 收集正式反馈
- 结算尾款
- 总结归档
- 维护客户关系

## 6. 质量标准

| 指标 | 标准 |
|------|------|
| 方案提交时效 | 带看后24小时内 |
| 合同准确率 | 100%（无遗漏/错误） |
| 活动执行满意度 | >95% |
| 客户投诉率 | <2% |
| 回头客比例 | >40% |

## 7. 应急预案

### 7.1 人数变动
- 增加：确认场地容量/餐饮准备能力
- 减少：按合同最低保证数执行

### 7.2 设备故障
- 备用设备待命
- 外部供应商快速响应

### 7.3 客户投诉
- 现场立即解决
- 权限内可赠送/减免
- 超权限立即上报

---

**培训要求**: 必须通过会议销售专项培训，包括场地知识、餐饮知识、视听设备知识  
**考核标准**: 会议销售额+客户满意度+活动执行零失误
