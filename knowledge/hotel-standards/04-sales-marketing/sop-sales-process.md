# 销售流程SOP (Sales Process SOP)

> **文档编号**: SM-SOP-001  
> **版本**: v1.0  
> **部门**: 市场销售部  
> **生效日期**: 2024-01-01

---

## 1. 目的

规范销售全流程，确保客户开发、跟进、签约、服务各环节标准化，提升销售业绩和客户满意度。

## 2. 适用范围

适用于市场销售部所有销售人员，包括散客销售、协议客户销售、会议销售、团队销售。

## 3. 销售全流程

```
线索获取 → 客户分级 → 需求分析 → 方案制定 → 报价谈判 → 签约成交 → 服务交付 → 客户维护 → 复购/续约
```

## 4. 客户开发阶段

### 4.1 线索来源

| 渠道 | 获取方式 | 责任人 |
|------|----------|--------|
| 官网询价 | 在线表单/400电话 | 预订部转销售 |
| OTA引流 | 携程/美团企业频道 | 渠道经理 |
| 转介绍 | 老客户/同行推荐 | 全员 |
| 主动开发 | 电话/拜访/展会 | 销售经理 |
| 集团分配 | 总部CRM分配 | 销售总监分配 |

### 4.2 线索录入（30分钟内）

**CRM录入要素**:
- [ ] 公司名称/个人姓名
- [ ] 联系方式（手机/座机/邮箱）
- [ ] 需求类型（住宿/会议/餐饮）
- [ ] 预计消费规模
- [ ] 预计使用时间
- [ ] 特殊需求备注
- [ ] 线索来源
- [ ] 分配销售人员

### 4.3 线索分级

**A级（高优先级）**:
- 需求明确，预算充足
- 决策人直接联系
- 1个月内有望成交
- 跟进频率：2-3天/次

**B级（中优先级）**:
- 有需求，在比较中
- 需培养关系
- 3个月内有望成交
- 跟进频率：每周/次

**C级（低优先级）**:
- 需求不明确
- 仅初步咨询
- 需长期培养
- 跟进频率：每月/次

## 5. 需求分析阶段

### 5.1 客户拜访前准备

**资料准备**:
- 酒店简介（电子版/纸质版）
- 房型介绍/价目表
- 会议室平面图/报价
- 餐饮菜单
- 成功案例
- 名片

**客户背景调查**:
- 公司规模/行业
- 历史消费记录（如有）
- 决策链条
- 竞品使用情况

### 5.2 拜访执行（首次拜访）

**开场（5分钟）**:
> "感谢您在百忙之中接待我。我是[酒店名]的销售经理[姓名]，今天来主要是了解贵公司的差旅/会议需求，看看我们酒店能为您提供什么样的服务。"

**需求探询（20分钟）**:
- 年差旅量/会议场次
- 预算范围
- 住店偏好（位置/档次/设施）
- 目前合作酒店及满意度
- 痛点/改进需求
- 决策流程/关键人

**酒店介绍（15分钟）**:
- 位置优势
- 设施设备（重点介绍匹配需求的）
- 服务特色
- 成功案例（同行业）

**初步方案（10分钟）**:
- 根据需求提供初步建议
- 留下资料
- 邀请参观

**结束（5分钟）**:
- 总结要点
- 确认下一步行动
- 约定下次联系时间
- 感谢

**拜访后（24小时内）**:
- 填写拜访记录
- 发送感谢邮件/短信
- 更新CRM状态
- 制定跟进计划

## 6. 方案制定阶段

### 6.1 方案内容

**企业客户协议方案**:
- 酒店简介
- 合作内容（客房/会议/餐饮）
- 价格政策
- 付款条款
- 服务承诺
- 双方权利义务

**会议/宴会方案**:
- 活动主题理解
- 场地推荐（平面图）
- 餐饮方案（菜单选择）
- 住宿安排
- 视听设备
- 布置方案
- 报价明细
- 备选方案

### 6.2 报价原则

**底价控制**:
- 不低于审批权限
- 特殊价格需报批
- 预留谈判空间（通常报价高于底价10-15%）

**报价单要素**:
- 有效期（通常7-15天）
- 价格明细
- 包含/不包含项目
- 付款条款
- 取消政策
- 联系人信息

## 7. 谈判与签约

### 7.1 谈判策略

**价格谈判**:
- 先谈价值，后谈价格
- 了解客户预算底线
- 让步要有条件（如增加量/长期合作）
- 小步让步，每次让步幅度递减

**常见异议处理**:

| 异议 | 应对话术 |
|------|----------|
| "价格太贵" | "我们的价格包含了XX服务，相比竞品其实更有价值。而且我们可以根据您的用量给予阶梯折扣。" |
| "考虑一下" | "没问题，您主要考虑哪些方面？我可以提供更多信息帮助您决策。" |
| "已经在用XX酒店" | "理解，切换供应商确实需要考量。我们可以先签一个小单试试，您对比看看我们的服务。" |

### 7.2 签约流程

**合同审核**:
- 销售经理初审
- 法务/财务审核（大额/特殊条款）
- 销售总监审批
- 总经理审批（超权限）

**签约**:
- 双方签字盖章
- 收取定金（如适用）
- 合同归档（原件财务，复印件销售）
- CRM更新合同信息
- 通知相关部门（前台/预订/财务）

## 8. 服务交付

### 8.1 内部交接（BEO制作）

**宴会通知单（BEO）要素**:
- 客户信息
- 活动日期/时间
- 场地安排
- 餐饮菜单
- 特殊要求
- 负责人联系
- 相关部门会签

**内部协调会**（活动前1-3天）:
- 前厅部：客房预留
- 餐饮部：菜单确认/备料
- 厨房：特殊要求
- 工程部：设备检查
- 安保部：停车/安全

### 8.2 活动执行跟进

**活动前1天**:
- 确认最终人数
- 检查场地布置
- 确认设备正常
- 与客户最终确认

**活动当天**:
- 提前到场检查
- 全程在场协调
- 及时处理突发情况
- 活动结束后送别客户

**活动后**:
- 收集反馈
- 结账收款
- 感谢客户
- 总结经验

## 9. 客户维护

### 9.1 分级维护

**VIP客户（年消费>50万）**:
- 每月拜访/通话
- 节假日问候/礼品
- 总经理亲自维护
- 专属优惠

**重点客户（年消费10-50万）**:
- 每季度拜访
- 季度消费报告
- 优先预留房间

**普通客户**:
- 定期邮件/短信维护
- 促销活动通知
- 年度满意度调查

### 9.2 客户回访

**首次消费后（3天内）**:
> "感谢您选择我们酒店，请问您对我们的服务还满意吗？有什么建议可以让我们做得更好？"

**定期回访**:
- 季度消费分析
- 新品/促销推荐
- 需求变化了解

## 10. 销售管理

### 10.1 销售例会

**频率**: 每周一上午
**内容**:
- 上周业绩回顾
- 本周工作计划
- 重点客户跟进
- 问题讨论

### 10.2 销售报告

**日报**: 当日拜访/通话记录，更新CRM
**周报**: 业绩进度，重点客户状态
**月报**: 业绩分析，市场动态，下月计划

### 10.3 销售工具

- CRM系统
- 销售手册
- 报价模板
- 合同模板
- 宣传物料

## 11. 质量标准

| 指标 | 标准 |
|------|------|
| 线索响应时间 | <30分钟 |
| 首次拜访完成率 | >80%（收到线索后1周内） |
| 方案提交时效 | 拜访后48小时内 |
| 成交转化率 | >30% |
| 客户满意度 | >90% |
| 复购率 | >60% |

---

**培训要求**: 新员工必须通过销售流程培训，老员工每年复训  
**考核标准**: 业绩指标+过程指标（拜访量/跟进及时性）
