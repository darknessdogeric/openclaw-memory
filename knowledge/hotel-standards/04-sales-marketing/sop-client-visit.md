# 客户拜访SOP (Client Visit SOP)

> **文档编号**: SM-SOP-002  
> **版本**: v1.0  
> **部门**: 市场销售部  
> **生效日期**: 2024-01-01

---

## 1. 目的

规范客户拜访流程，提升拜访效率和专业形象，建立良好客户关系。

## 2. 适用范围

适用于所有客户拜访活动，包括新客户开发拜访、老客户维护拜访、会议需求洽谈。

## 3. 拜访类型

| 类型 | 目的 | 时长 | 频率 |
|------|------|------|------|
| 初次拜访 | 建立联系、了解需求 | 45-60分钟 | 1次 |
| 需求深挖 | 详细需求调研 | 30-45分钟 | 1-2次 |
| 方案汇报 | 提交方案、答疑 | 30-60分钟 | 1-2次 |
| 关系维护 | 保持联络、挖掘机会 | 15-30分钟 | 每季度 |
| 问题解决 | 处理投诉/特殊需求 | 视情况 | 按需 |

## 4. 拜访前准备

### 4.1 信息准备（T-1日）

**客户信息**:
- [ ] 公司基本情况（规模/行业/业务）
- [ ] 历史合作记录（如有）
- [ ] 关键决策人信息
- [ ] 竞品使用情况
- [ ] 潜在需求分析

**拜访目标设定**:
- [ ] 本次拜访核心目标（如：建立联系/获取需求/促成签约）
- [ ] 最低目标（如：获取决策人联系方式）
- [ ] 理想目标（如：现场签约）

**资料准备**:
- [ ] 酒店简介（最新版）
- [ ] 房型照片/价目表
- [ ] 会议场地资料
- [ ] 餐饮菜单
- [ ] 成功案例（同行业优先）
- [ ] 名片（充足）
- [ ] 笔记本/笔
- [ ] 合同/协议（如需要）

**形象准备**:
- [ ] 职业装（西装/套装）
- [ ] 仪容整洁
- [ ] 口气清新
- [ ] 手机静音

### 4.2 预约确认（T-1日）

**电话预约话术**:
> "[客户姓氏]经理您好，我是[酒店名]的销售经理[姓名]。想和您约个时间拜访，介绍一下我们酒店的服务，看看能否为贵公司的差旅/会议提供支持。您看这周[时间]方便吗？"

**确认要素**:
- 具体日期时间
- 拜访地点
- 拜访时长
- 对方参与人员
- 联系方式（防找不到）

**预约后**:
- 发送确认短信/邮件
- 设置日程提醒
- 规划路线（提前查交通）

## 5. 拜访执行

### 5.1 到达前（T-15分钟）

- 再次确认地址
- 检查资料是否齐全
- 整理仪容
- 手机静音
- 提前10-15分钟到达

### 5.2 开场（5分钟）

**寒暄**:
> "[客户姓氏]经理，您好！非常感谢您百忙之中抽出时间。我是[酒店名]的[姓名]，这是我的名片。"

**破冰话题**（可选）:
- 办公环境
- 行业动态
- 近期热点

**转入正题**:
> "今天来主要是想了解一下贵公司在差旅/会议方面的需求，看看我们酒店能为贵公司提供什么样的服务。"

### 5.3 需求探询（15-20分钟）

**SPIN提问法**:

**S - 情境问题（Situation）**:
- "贵公司每年的差旅量大概多少？"
- "一般是什么类型的会议/活动？"
- "目前主要使用哪些酒店？"

**P - 问题问题（Problem）**:
- "目前合作酒店有什么让您不满意的地方吗？"
- "差旅管理方面有什么痛点？"
- "会议组织中遇到什么困难？"

**I - 暗示问题（Implication）**:
- "这些问题对贵公司的影响大吗？"
- "如果不解决会有什么后果？"

**N - 需求-效益问题（Need-payoff）**:
- "如果能解决XX问题，对贵公司会有什么帮助？"
- "您希望理想的合作酒店是什么样的？"

**倾听技巧**:
- 目光接触
- 点头回应
- 记录要点
- 适时追问
- 不打断

### 5.4 酒店介绍（10-15分钟）

**FAB介绍法**:

**F - 特点（Feature）**:
> "我们酒店位于[位置]，拥有[设施]..."

**A - 优势（Advantage）**:
> "这意味着您可以[便利/节省成本]..."

**B - 利益（Benefit）**:
> "对贵公司来说，这将带来[具体好处]..."

**重点突出匹配客户需求的**: 根据探询到的需求，重点介绍相关设施和服务。

### 5.5 方案初步沟通（5-10分钟）

**根据需求提供建议**:
- "根据您刚才提到的需求，我建议..."
- "我们有成功案例，类似贵公司这样的企业..."

**邀请参观**:
> "如果方便的话，邀请您和您的同事来我们酒店实地参观，我可以安排专人接待。"

### 5.6 结束与约定（5分钟）

**总结要点**:
> "今天和您沟通下来，我了解到贵公司的主要需求是...我们酒店在...方面可以满足。"

**下一步行动**:
> "接下来我会根据您的需求准备一份详细的方案和报价，[时间]前发给您。您看可以吗？"

**感谢**:
> "非常感谢您今天抽出宝贵时间，期待能和贵公司合作。"

**握手告别**:
- 起身
- 整理物品
- 握手
- 目送离开

## 6. 拜访后跟进

### 6.1 拜访记录（2小时内）

**CRM录入**:
- 拜访时间/地点
- 参与人员
- 沟通要点
- 客户需求
- 下一步计划
- 预计成交时间
- 成交金额预测

### 6.2 感谢信（24小时内）

**邮件模板**:
```
主题：感谢[公司名称][客户姓氏]经理的接待 - [酒店名]

尊敬的[客户姓氏]经理：

您好！

非常感谢您昨日百忙之中抽出时间与我见面。通过交流，我对贵公司的需求有了更深入的了解。

根据我们的沟通，我已开始准备[具体内容]，预计[时间]前发送给您。

附件是我们酒店的详细资料，供您参考。

如有任何问题，欢迎随时联系我。

再次感谢您的接待，期待与贵公司的合作！

祝商祺！

[姓名]
[职位]
[酒店名]
[手机]
[邮箱]
```

### 6.3 内部沟通

**重要客户**:
- 向销售总监汇报拜访情况
- 申请资源支持（如需要）
- 协调内部配合

## 7. 拜访质量检查

### 7.1 自我检查清单

- [ ] 是否准时到达
- [ ] 形象是否专业
- [ ] 是否了解客户需求
- [ ] 是否清晰介绍酒店优势
- [ ] 是否达成拜访目标
- [ ] 是否确定下一步行动
- [ ] 是否记录完整
- [ ] 是否及时跟进

### 7.2 销售总监检查

**每周抽查**:
- 拜访记录完整性
- 客户需求把握准确性
- 跟进计划合理性

## 8. 特殊情况处理

### 8.1 客户临时取消

- 保持礼貌
- 询问改期时间
- 发送改期确认

### 8.2 关键人不在

- 与替代人员建立联系
- 获取关键人联系方式
- 简要介绍，争取单独拜访关键人

### 8.3 客户态度冷淡

- 保持专业和热情
- 缩短拜访时间
- 礼貌告辞，后续通过其他方式培养

---

**培训要求**: 新员工需通过拜访技巧培训和角色扮演考核  
**考核标准**: 拜访量（每周≥8次）+ 拜访质量评分
