# 价格管理政策 (Rate Management P&P)

> **文档编号**: SM-PP-001  
> **版本**: v1.0  
> **部门**: 市场销售部  
> **生效日期**: 2024-01-01

---

## 1. 目的

规范酒店定价策略和折扣审批流程，确保收益最大化，维护品牌形象。

## 2. 适用范围

适用于所有房型、所有渠道的定价管理。

## 3. 价格体系

### 3.1 价格层级

```
门市价 (Rack Rate)
    │
    ├─ 散客价 (Walk-in)
    ├─ 会员价 (Member Rate)
    ├─ 协议价 (Corporate Rate)
    ├─ 团队价 (Group Rate)
    ├─ OTA价 (Commissionable Rate)
    └─ 促销价 (Promotional Rate)
```

### 3.2 折扣权限

| 折扣幅度 | 审批人 | 适用场景 |
|----------|--------|----------|
| ≤10% | 销售经理 | 协议客户续约 |
| 10-20% | 销售总监 | 大客户/长住 |
| 20-30% | 总经理 | 特殊情况 |
| >30% | 需报备业主 | 极端情况 |

### 3.3 最低价政策

**BAR (Best Available Rate)**:
- 所有渠道公开价格不得低于BAR
- 会员价可比BAR低5-10%
- 协议价可比BAR低10-20%

**价格一致性**:
- 官网价格必须是最优价格
- OTA价格不得高于官网
- 违反价格一致性将被处罚

## 4. 渠道定价政策

| 渠道 | 价格策略 | 佣金 | 说明 |
|------|----------|------|------|
| 官网直销 | 最优价格 | 0% | 优先推广 |
| 协议客户 | 阶梯折扣 | 0% | 年度用量决定折扣 |
| 携程 | 挂牌价 | 10-15% | 保持竞争力 |
| 美团 | 促销价 | 8-12% | 周末/本地推广 |
| 飞猪 | 会员价 | 10-12% | 阿里生态 |
| 团队 | 打包价 | 视情况 | 淡季促销 |

## 5. 促销政策

### 5.1 促销审批

**促销活动需审批内容**:
- 促销价格
- 促销时段
- 适用房型
- 促销渠道
- 预期效果

**审批流程**:
销售经理提案 → 收益经理审核 → 销售总监审批 → GM备案

### 5.2 促销类型

| 类型 | 适用时机 | 折扣范围 |
|------|----------|----------|
| 早鸟价 | 提前预订 | 5-15% |
| 连住优惠 | 3晚以上 | 10-20% |
| 周末套餐 | 周五-周日 | 含餐/体验 |
| 尾房促销 | 当日18:00后 | 20-30% |
| 会员专享 | 会员日 | 额外5% |

## 6. 价格调整政策

### 6.1 日常调整

**调整权限**:
- 收益经理：每日价格调整（±10%）
- 销售总监：重大活动定价
- GM：特殊事件定价

**调整依据**:
- 预订进度
- 竞争价格
- 市场需求
- 历史数据

### 6.2 动态定价

**触发条件**:
- 预订率>80%：涨价
- 预订率<50%：促销
- 竞争对手降价：跟进
- 重大活动：溢价

## 7. 职责分工

| 角色 | 职责 |
|------|------|
| 收益经理 | 日常定价、动态调价、数据分析 |
| 销售总监 | 大客户价格、促销审批 |
| 销售经理 | 协议价格谈判、促销执行 |
| 前台经理 | 现场价格控制、Walk-in定价 |

## 8. 违规处罚

| 违规行为 | 处罚 |
|----------|------|
| 未经审批低价销售 | 差价由责任人承担 |
| 价格泄露给竞争对手 | 纪律处分 |
| 恶意低价竞争 | 警告/降职 |
