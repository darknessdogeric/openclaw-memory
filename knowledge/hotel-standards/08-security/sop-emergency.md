# 应急响应SOP (Emergency Response SOP)

> **文档编号**: SEC-SOP-001  
> **版本**: v1.0  
> **部门**: 安保部  
> **生效日期**: 2024-01-01

---

## 1. 目的

规范各类突发事件的应急响应流程，保障宾客和员工生命安全，减少财产损失。

## 2. 适用范围

适用于酒店内发生的火灾、治安、医疗、自然灾害等突发事件。

## 3. 火灾应急预案

### 3.1 发现火情

**任何人发现火情**:
1. 立即呼叫："着火了！"
2. 就近按下手动报警器
3. 拨打内部报警电话（XXXX）
4. 尝试初期灭火（如火势可控）
5. 组织人员疏散

### 3.2 安保响应

**3分钟内**:
- 确认火情位置
- 启动消防广播
- 组织初期灭火
- 启动排烟系统
- 拨打119

**5分钟内**:
- 组织人员疏散
- 关闭非消防电源
- 启动消防泵
- 引导消防车

### 3.3 疏散程序

**疏散原则**:
- 宾客优先
- 就近疏散
- 不使用电梯
- 清点人数

**疏散路线**:
- 按疏散指示标志
- 走楼梯
- 到集合点
- 禁止返回

**疏散话术**:
> "请不要惊慌，跟随我，走楼梯，不要使用电梯。"

### 3.4 集合点

| 区域 | 集合点 |
|------|--------|
| 客房区 | 酒店正门广场 |
| 餐厅 | 停车场 |
| 后区 | 员工通道外 |

### 3.5 灭火原则

- 救人第一
- 先控制后消灭
- 先重点后一般

## 4. 治安事件应急

### 4.1 盗窃案件

**发现盗窃**:
1. 保护现场
2. 报告安保经理
3. 询问当事人
4. 查看监控
5. 报警（如需要）
6. 记录备案

### 4.2 打架斗殴

**处置流程**:
1. 立即制止
2. 分开当事人
3. 控制局面
4. 如有伤情，呼叫医疗
5. 报警
6. 记录证人信息

### 4.3 可疑人员

**发现可疑人员**:
1. 暗中观察
2. 报告监控中心
3. 跟踪监视
4. 必要时询问
5. 如有异常，报警

## 5. 医疗急救

### 5.1 客人突发疾病

**响应流程**:
1. 立即到现场
2. 判断意识/呼吸
3. 呼叫120
4. 实施急救（持证人员）
5. 通知前台联系家属
6. 保护隐私

### 5.2 外伤处理

**轻微外伤**:
- 清洁伤口
- 消毒
- 包扎
- 建议就医

**严重外伤**:
- 止血
- 固定
- 呼叫120
- 不随意移动

### 5.3 心肺复苏 (CPR)

**适用**: 无呼吸/无心跳

**步骤**:
1. 确认安全
2. 判断意识
3. 呼叫帮助
4. 开放气道
5. 人工呼吸（2次）
6. 胸外按压（30次）
7. 循环进行（30:2）
8. 使用AED（如有）

## 6. 自然灾害

### 6.1 地震

**震时**:
- 就地避险
- 躲避在坚固家具旁
- 远离玻璃/高处物品
- 不乘电梯

**震后**:
- 检查伤亡
- 检查设施损坏
- 疏散到开阔地
- 关闭燃气/电源

### 6.2 暴雨/洪水

**预防措施**:
- 检查排水系统
- 准备沙袋
- 转移低处物品
- 准备应急物资

**应对措施**:
- 启动排水泵
- 封堵入口
- 疏散低层客人
- 联系救援

### 6.3 台风

**预防措施**:
- 加固门窗
- 移走室外物品
- 检查屋顶
- 储备物资

**台风期间**:
- 停止营业
- 人员避险
- 停止电梯
- 备用发电

## 7. 其他紧急情况

### 7.1 电梯困人

**响应**:
1. 安抚被困人员
2. 通知工程部
3. 联系电梯维保
4. 必要时拨打119
5. 等待专业救援

**注意**: 不要强行撬门

### 7.2 燃气泄漏

**响应**:
1. 立即关闭总阀
2. 开窗通风
3. 禁止明火/电器
4. 疏散人员
5. 联系燃气公司
6. 排查泄漏点

### 7.3 食物中毒

**响应**:
1. 保留食物样本
2. 送医救治
3. 报告卫生部门
4. 封存厨房
5. 配合调查
