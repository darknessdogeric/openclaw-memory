# 安保巡逻SOP (Security Patrol SOP)

> **文档编号**: SEC-SOP-002  
> **版本**: v1.0  
> **部门**: 安保部  
> **生效日期**: 2024-01-01

---

## 1. 目的

规范安保巡逻流程，及时发现和处理安全隐患，确保酒店人员和财产安全。

## 2. 适用范围

适用于安保部所有巡逻岗位。

## 3. 巡逻类型

| 类型 | 频率 | 范围 | 重点 |
|------|------|------|------|
| 固定岗 | 持续 | 大堂/门口 | 人员进出监控 |
| 定时巡逻 | 每小时 | 全酒店 | 全面检查 |
| 重点巡逻 | 每2小时 | 机房/仓库/财务室 | 要害部位 |
| 夜间巡逻 | 每30分钟 | 客房区/外围 | 安全防范 |

## 4. 巡逻路线

### 4.1 日间巡逻路线

```
起点：安保室
↓
大堂（检查消防/监控/可疑人员）
↓
餐厅/厨房（检查燃气/消防/食品安全）
↓
会议室/宴会厅（检查设备/门窗）
↓
后台区（工程部/员工通道）
↓
外围（停车场/周边）
↓
返回安保室
```

### 4.2 夜间巡逻路线

```
起点：安保室
↓
大堂（确认关闭/锁好）
↓
各楼层客房区（检查门窗/可疑人员）
↓
消防楼梯（检查通道畅通）
↓
机房/配电室（检查设备运行）
↓
外围/停车场（检查可疑车辆/人员）
↓
返回安保室
```

## 5. 巡逻要点

### 5.1 检查内容

**消防安全**:
- [ ] 消防设施完好（灭火器/消火栓/烟感）
- [ ] 消防通道畅通
- [ ] 应急照明正常
- [ ] 防火门关闭

**治安安全**:
- [ ] 无可疑人员
- [ ] 无可疑物品
- [ ] 门窗锁闭
- [ ] 贵重物品安全

**设备安全**:
- [ ] 机房运行正常
- [ ] 无异常声音/气味
- [ ] 设备房锁闭
- [ ] 配电室正常

**其他**:
- [ ] 照明正常
- [ ] 空调运行
- [ ] 卫生状况
- [ ] 设施完好

### 5.2 重点部位

**每小时必查**:
- 大堂
- 门口
- 监控室

**每2小时必查**:
- 配电室
- 锅炉房
- 水泵房
- 厨房
- 财务室
- 仓库

**夜间加强**:
- 客房区走廊
- 消防楼梯
- 停车场
- 外围围墙

## 6. 巡逻程序

### 6.1 巡逻前

- 检查装备（对讲机/手电筒/警棍/记录本）
- 确认通讯正常
- 了解当班情况
- 查看上一班次记录

### 6.2 巡逻中

**巡逻方式**:
- 步行巡逻（主要）
- 定点签到（打卡点）
- 视频监控辅助

**注意事项**:
- 保持警觉
- 观察细致
- 记录完整
- 及时处理异常

**发现异常处理**:
1. 评估危险程度
2. 能处理：立即处理
3. 不能处理：立即报告，请求支援
4. 保护现场
5. 记录详情

### 6.3 巡逻后

- 填写巡逻记录
- 报告异常情况
- 交接班说明
- 装备归位

## 7. 巡逻记录

### 7.1 记录内容

**巡逻日志**:
| 项目 | 内容 |
|------|------|
| 日期/时间 | 精确到分钟 |
| 巡逻人员 | 姓名/工号 |
| 巡逻路线 | 经过区域 |
| 检查情况 | 正常/异常 |
| 异常详情 | 时间/地点/情况/处理 |
| 签到点 | 打卡时间 |
| 交接事项 | 待办事项 |

### 7.2 电子巡更（如有）

- 按点位打卡
- 记录自动上传
- 异常及时上传照片

## 8. 特殊情况处理

### 8.1 发现火情

1. 立即报警（119+内部）
2. 初期灭火（如火势小）
3. 组织疏散
4. 引导消防车

### 8.2 发现可疑人员

1. 暗中观察
2. 通知监控中心跟踪
3. 必要时上前询问
4. 有异常情况立即控制并报警

### 8.3 发现盗窃

1. 保护现场
2. 立即报告
3. 查看监控
4. 配合调查

### 8.4 客人醉酒/闹事

1. 劝阻制止
2. 通知大堂经理
3. 必要时报警
4. 保护其他客人安全

## 9. 装备管理

### 9.1 巡逻装备

- 对讲机
- 手电筒
- 警棍（防卫）
- 巡逻记录本
- 巡更棒（如有）

### 9.2 装备使用

- 正确使用
- 爱护保管
- 交接清点
- 故障报修

## 10. 质量标准

| 指标 | 标准 |
|------|------|
| 巡逻频率 | 按标准执行，无遗漏 |
| 签到率 | 100% |
| 异常情况发现率 | 100% |
| 记录完整率 | 100% |
| 安全事故 | 0 |

---

**培训要求**: 必须通过巡逻路线培训、应急处理培训  
**考核标准**: 巡逻签到率+异常情况发现及时性+记录完整性
