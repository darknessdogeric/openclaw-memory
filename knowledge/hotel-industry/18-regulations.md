# 18 政策法规 - 合规指南

> 开业证照 · 消防法规 · 环保要求 · 劳动法规 · 安全责任

---

## 一、开业必备证照

### 1.1 六大核心证照

| 证照 | 主管部门 | 办理时机 | 关键要求 |
|------|----------|----------|----------|
| **营业执照** | 市场监督管理局 | 筹建初期 | 经营范围含住宿 |
| **特种行业许可证** | 公安局 | 开业前 | 治安条件达标 |
| **卫生许可证** | 卫健委 | 开业前 | 卫生设施、检测报告 |
| **消防验收合格证** | 消防救援机构 | 开业前 | 消防系统、疏散通道 |
| **食品经营许可证** | 市场监管局 | 开业前（如有餐饮） | 厨房设施、人员健康证 |
| **税务登记** | 税务局 | 开业前 | 税种核定、发票申领 |

### 1.2 证照办理流程

```
Step 1: 工商注册（营业执照）
   ↓
Step 2: 消防设计审核
   ↓
Step 3: 装修施工
   ↓
Step 4: 消防验收
   ↓
Step 5: 卫生许可申请
   ↓
Step 6: 特种行业许可
   ↓
Step 7: 食品经营许可（如有餐饮）
   ↓
Step 8: 税务登记
   ↓
正式开业
```

## 二、消防法规

### 2.1 核心法规

**《消防法》**要点:
- 消防安全责任制
- 消防设施维护
- 疏散通道畅通
- 定期消防演练

**《建筑设计防火规范》**（GB50016）:
- 建筑分类
- 防火分区
- 疏散距离
- 消防设施配置

### 2.2 酒店消防要求

**设施配置**:
| 设施 | 要求 |
|------|------|
| 自动喷淋 | 客房、走廊全覆盖 |
| 火灾报警 | 烟感、温感、手报 |
| 消火栓 | 每50米一个 |
| 灭火器 | 每50㎡配置 |
| 应急照明 | 断电自动开启 |
| 疏散指示 | 清晰、不遮挡 |

**管理要求**:
- 每日防火巡查
- 每月设施检查
- 每半年演练
- 24小时消防控制室值班

### 2.3 民宿消防特规

**住宿业消防新规（2019）**:
- 客房数≤14间、面积≤800㎡：简化审批
- 超过标准：按酒店标准执行
- 必须配置：喷淋/报警、灭火器、应急照明

## 三、环保要求

### 3.1 环评要求

**适用情况**:
- 新建酒店：需环境影响评价
- 改造酒店：视规模可能需备案
- 一般规模：环境影响登记表备案

**环保设施**:
| 项目 | 要求 |
|------|------|
| 污水处理 | 接入市政管网或处理达标 |
| 油烟处理 | 净化设备、高空排放 |
| 噪声控制 | 边界达标 |
| 垃圾分类 | 按要求分类收集 |

### 3.2 排污许可

**办理条件**:
- 向环境部门申请
- 提交排污方案
- 定期监测报告

---

## 四、劳动法规

### 4.1 核心法规

**《劳动合同法》**:
- 书面合同
- 试用期规定
- 解除条件
- 经济补偿

**《社会保险法》**:
- 五险（养老/医疗/失业/工伤/生育）
- 强制缴纳
- 基数与比例

### 4.2 酒店行业特殊规定

**工时制度**:
| 岗位 | 工时制 | 说明 |
|------|--------|------|
| 管理岗 | 标准工时 | 8小时/天 |
| 前台 | 综合工时 | 24小时轮班 |
| 客房 | 综合工时 | 弹性排班 |
| 工程 | 综合工时 | 紧急响应 |

**加班规定**:
- 平时：1.5倍
- 周末：2倍
- 节假日：3倍

### 4.3 员工权益保障

**必备保障**:
- 劳动合同
- 五险一金
- 带薪年假
- 劳动保护
- 职业培训

## 五、安全责任

### 5.1 治安责任

**《旅馆业治安管理办法》**:
- 实名登记（一人一证）
- 访客管理
- 可疑情况报告
- 监控保存（30天+）

### 5.2 食品安全

**《食品安全法》**:
- 食品经营许可证
- 从业人员健康证
- 食材索证索票
- 48小时留样
- 明厨亮灶

### 5.3 特种设备

**电梯安全**:
- 注册登记
- 定期检验
- 维保合同
- 应急预案

**锅炉/压力容器**（如有）:
- 定期检验
- 持证操作
- 安全附件完好

## 六、合规检查清单

### 6.1 开业前检查

| 类别 | 检查项 | 状态 |
|------|--------|------|
| 证照 | 6大证照齐全 | ☐ |
| 消防 | 验收合格、设施完好 | ☐ |
| 卫生 | 检测合格、许可证照 | ☐ |
| 环保 | 备案/许可完成 | ☐ |
| 安全 | 制度上墙、培训完成 | ☐ |
| 劳动 | 合同模板、社保开户 | ☐ |

### 6.2 日常合规检查

**每日**:
- 消防巡查
- 安全监控
- 实名登记

**每月**:
- 设施检查
- 卫生检查
- 培训记录

**每年**:
- 证照年检
- 消防演练
- 环保检测
- 电梯年检

## 七、违规风险与处罚

| 违规行为 | 处罚措施 | 风险等级 |
|----------|----------|----------|
| 无证经营 | 罚款、停业、取缔 | 高 |
| 消防不合格 | 罚款、停业整顿 | 高 |
| 未实名登记 | 罚款、治安处罚 | 中 |
| 未缴社保 | 补缴+滞纳金 | 中 |
| 食品安全问题 | 罚款、吊销许可 | 高 |
