## 一、万豪国际 (Marriott International)

### 1.1 集团概况

**规模数据（2024）**:
- 酒店数量：8,700+
- 客房数量：160万+
- 品牌数量：30个
- 市值：~$600亿

### 1.2 品牌矩阵

| 档次 | 品牌 | 定位 |
|------|------|------|
| **奢华** | 丽思卡尔顿、瑞吉、宝格丽、W、JW万豪、艾迪逊 | 极致服务、设计先锋 |
| **高端** | 万豪、喜来登、艾美、威斯汀、万丽 | 商务首选、会议强项 |
| **精选** | 万怡、福朋、雅乐轩、万枫、SpringHill | 精选服务、性价比 |
| **长住** | 万豪行政公寓、Residence Inn、TownePlace | 长住客群、居家设施 |

### 1.3 并购史

| 时间 | 交易 | 金额 | 战略意义 |
|------|------|------|----------|
| 2016 | 收购喜达屋 | $136亿 | 全球最大酒店集团 |
| 1997 | 收购丽思卡尔顿 | 未披露 | 进入奢华市场 |
| 1995 | 收购丽嘉 | 未披露 | 强化高端布局 |

### 1.4 中国战略

**布局重点**:
- 一线城市：奢华品牌全覆盖
- 新一线：高端+精选扩张
- 下沉市场：万枫、万怡

**轻资产转型**:
- 管理合同占比：70%+
- 特许经营占比：20%+
- 自有物业：<10%

## 二、希尔顿集团 (Hilton)

### 2.1 集团概况

**规模数据（2024）**:
- 酒店数量：7,500+
- 客房数量：110万+
- 品牌数量：19个
- 市值：~$450亿

### 2.2 品牌矩阵

| 档次 | 品牌 | 定位 |
|------|------|------|
| **奢华** | 华尔道夫、康莱德、LXR | 传奇服务、地标建筑 |
| **高端** | 希尔顿、格芮精选、Curio | 经典商务、在地特色 |
| **中端** | 希尔顿花园、欢朋、惠庭 | 可靠品质、投资回报 |
| **经济** | Tru、Motto | 年轻、简约 |

### 2.2 核心优势

**会员体系 (Hilton Honors)**:
- 会员数量：1.5亿+
- 贡献间夜：60%+
- 直销占比：行业领先

**数字化创新**:
- 数字钥匙（手机开门）
- 直连预订
- 智能客房

### 2.3 中国战略

**增长策略**:
- 品牌下沉：欢朋快速扩张
- 特许经营：加速中国市场
- 生活方式：Curio、Canopy引入

## 六、中国市场竞争格局

### 6.1 国际品牌在华布局

```
┌─────────────────────────────────────────────────────────────┐
│                  国际品牌在华战略层级                         │
├─────────────────┬───────────────────────────────────────────┤
│     奢华层      │ 丽思、安缦、宝格丽、四季、半岛             │
│                 │ 布局：一线城市核心（10-20家/品牌）          │
├─────────────────┼───────────────────────────────────────────┤
│     高端层      │ 万豪、洲际、希尔顿、凯悦                   │
│                 │ 布局：一二线城市（200-500家/集团）          │
├─────────────────┼───────────────────────────────────────────┤
│    中高端层     │ 美居、诺富特、假日、皇冠                   │
│                 │ 布局：与本土品牌激烈竞争                    │
├─────────────────┼───────────────────────────────────────────┤
│     中端层      │ 智选、欢朋、美居、诺富特                   │
│                 │ 布局：被华住/锦江压制                       │
├─────────────────┼───────────────────────────────────────────┤
│     下沉市场    │ 几乎空白（本土品牌主导）                    │
└─────────────────┴───────────────────────────────────────────┘
```

### 6.2 竞争趋势

**国际品牌挑战**:
- 中端市场被本土挤压
- 下沉市场难渗透
- 管理成本高

**国际品牌机会**:
- 高端市场仍有空间
- 品牌升级需求
- 出境游恢复后优势
