## 一、收入结构模型

### 1.1 酒店收入构成

```
┌─────────────────────────────────────────────────────────────┐
│                    酒店收入结构                              │
├─────────────────────────────────────────────────────────────┤
│  客房收入      ████████████████████   60-70%               │
│  餐饮收入      ██████████            20-30%               │
│  其他收入      ████                   10-15%               │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 收入明细分解

**客房收入（核心）**:
```
客房收入 = 可售客房数 × Occ × ADR × 365

示例：100间房 × 80% × ¥400 × 365 = ¥1,168万/年
```

**餐饮收入（重要补充）**:
| 业态 | 占比 | 毛利率 | 适用场景 |
|------|------|--------|----------|
| 早餐 | 30% | 70% | 全档次标配 |
| 正餐 | 40% | 60% | 高端/度假 |
| 宴会 | 20% | 75% | 会议酒店 |
| 酒吧 | 10% | 80% | 高端酒店 |

**其他收入**: 会议室、洗衣、停车、迷你吧、零售、佣金

## 二、成本结构分析

### 2.1 成本分类

**固定成本（60-70%）**:
| 成本项 | 占比 | 特征 |
|--------|------|------|
| 人力成本 | 25-35% | 随通胀逐年上涨 |
| 租金/折旧 | 20-30% | 长期锁定或分摊 |
| 能耗 | 8-12% | 季节性波动 |
| 维保/其他 | 5-8% | 预防性支出 |

**变动成本（30-40%）**:
| 成本项 | 占比 | 特征 |
|--------|------|------|
| 餐饮成本 | 10-15% | 与餐饮收入挂钩 |
| 客房易耗 | 3-5% | 与入住率挂钩 |
| 营销费用 | 5-10% | 可控、可调节 |
| OTA佣金 | 3-8% | 与渠道结构挂钩 |

### 2.2 GOP计算

```
营业收入
- 营业成本
- 人力成本
- 能耗
- 物料消耗
- 维修保养
- 营销费用
────────────────
= GOP（经营毛利）

GOP率 = GOP ÷ 营业收入
目标：中高端酒店 35%+，经济型酒店 40%+
```

## 三、投资回报单店模型

### 3.1 中端酒店投资模型（示例）

**基础参数**:
| 参数 | 数值 |
|------|------|
| 客房数 | 100间 |
| 单房投资 | 12万 |
| 总投资 | 1,200万 |
| 目标ADR | ¥380 |
| 目标Occ | 80% |
| 目标GOP率 | 38% |

**收入测算**:
```
客房收入 = 100 × 80% × ¥380 × 365 = ¥1,109万
餐饮收入 = ¥1,109万 × 25% = ¥277万
其他收入 = ¥1,109万 × 10% = ¥111万
────────────────────────────────
总收入 = ¥1,497万
```

**成本测算**:
```
人力成本 (30%) = ¥449万
租金/折旧 (25%) = ¥374万
能耗 (10%) = ¥150万
其他运营 (22%) = ¥329万
────────────────────────────────
总成本 = ¥1,302万
```

**回报测算**:
```
GOP = ¥1,497万 - ¥1,302万 = ¥195万
GOP率 = 13%（偏低，需优化）

若GOP率达到38%：
GOP = ¥1,497万 × 38% = ¥569万
投资回收期 = 1,200万 ÷ 569万 = 2.1年（过快，需验证参数）
```

### 3.2 不同档次对比

| 档次 | 单房投资 | RevPAR目标 | GOP率 | 回收期 |
|------|----------|------------|-------|--------|
| 经济型 | 6-8万 | ¥150-200 | 40-45% | 3-4年 |
| 中端 | 10-12万 | ¥250-350 | 35-40% | 4-6年 |
| 中高端 | 13-16万 | ¥350-500 | 30-35% | 5-7年 |
| 高端 | 18-25万 | ¥500-800 | 25-30% | 6-8年 |

## 四、敏感性分析

### 4.1 单因素变动影响

**ADR变动影响**（基准ADR¥400，Occ 80%）:

| ADR变动 | 新ADR | 年收入变动 | GOP率变化 |
|---------|-------|------------|-----------|
| +20% | ¥480 | +234万 | +8% |
| +10% | ¥440 | +117万 | +4% |
| 基准 | ¥400 | - | 38% |
| -10% | ¥360 | -117万 | -4% |
| -20% | ¥320 | -234万 | -8% |

**Occ变动影响**（基准80%）:

| Occ变动 | 新Occ | 年收入变动 | GOP率变化 |
|---------|-------|------------|-----------|
| +10pt | 90% | +146万 | +5% |
| +5pt | 85% | +73万 | +2.5% |
| 基准 | 80% | - | 38% |
| -5pt | 75% | -73万 | -2.5% |
| -10pt | 70% | -146万 | -5% |

### 4.2 双因素矩阵

```
           ADR变动
          -20%  -10%   0   +10%  +20%
        ┌─────┬─────┬─────┬─────┬─────┐
  +10pt │     │     │     │     │最优 │
   +5pt │     │     │基准 │     │     │
Occ  0  │     │     │ 38% │     │     │
   -5pt │     │     │     │     │     │
  -10pt │最差 │     │     │     │     │
        └─────┴─────┴─────┴─────┴─────┘
```

### 4.3 盈亏平衡点分析

**盈亏平衡ADR**（假设Occ 80%不变）:
```
固定成本 = 人力 + 租金 + 基础能耗 = 800万
变动成本率 = 30%
盈亏平衡ADR = 固定成本 ÷ [客房数 × Occ × 365 × (1-变动成本率)]
            = 800万 ÷ [100 × 80% × 365 × 70%]
            = ¥391
```

**盈亏平衡Occ**（假设ADR ¥400不变）:
```
盈亏平衡Occ = 固定成本 ÷ [客房数 × ADR × 365 × (1-变动成本率)]
            = 800万 ÷ [100 × 400 × 365 × 70%]
            = 78%
```

## 五、现金流管理

### 5.1 月度现金流模型

```
期初现金
+ 营业收入（现金收款）
- 运营成本支付
- 租金/贷款支付
- 资本支出
────────────────
= 期末现金
```

### 5.2 现金储备要求

| 场景 | 现金储备建议 |
|------|--------------|
| 稳定运营 | 2-3个月运营支出 |
| 爬坡期 | 4-6个月运营支出 |
| 重大改造 | 预留改造资金+3个月运营支出 |

## 六、财务健康度指标

### 6.1 核心监控指标

| 指标 | 健康区间 | 预警阈值 | 优化方向 |
|------|----------|----------|----------|
| GOP率 | 35-45% | <30% | 成本控制/收入提升 |
| 人力占比 | 25-30% | >35% | 人房比优化 |
| 能耗占比 | 8-12% | >15% | 节能改造 |
| OTA占比 | <30% | >50% | 直销建设 |
| 应收账款天数 | <30天 | >45天 | 账期管理 |

### 6.2 债务覆盖率

```
债务覆盖率 = GOP ÷ 债务本息

健康标准：>1.25
预警标准：<1.0（无法覆盖债务）
```

## 七、投资决策框架

### 7.1 投资前必算

1. **投资额**: 单房造价 × 客房数 + 开办费 + 预备金
2. **收入预测**: RevPAR目标 × 365 × 客房数
3. **成本预估**: 固定成本 + 变动成本率 × 收入
4. **GOP测算**: 收入 - 成本
5. **回收期**: 投资额 ÷ 年GOP

### 7.2 投资可行性评估

| 评估维度 | 通过标准 | 风险等级 |
|----------|----------|----------|
| 静态回收期 | <6年 | 高 |
| 动态IRR | >15% | 高 |
| 盈亏平衡Occ | <70% | 中 |
| 敏感性 | ADR±10%不亏损 | 中 |
