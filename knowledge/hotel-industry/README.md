## 🎯 投资决策链使用指南

**完整投资决策路径**:
```
机会发现 → 可行性研究 → 深度调研 → 财务测算 → 尽职调查
    ↓
交易谈判 → 交割筹建 → 运营监控 → 退出执行
```

**关键决策节点**:
1. **初筛决策**（27号文件节点1 checklist）
2. **可行性决策**（26号文件评分卡）
3. **投资决议**（27号文件投委会材料）
4. **交割前核查**（27号文件交割清单）
5. **退出决策**（11号文件红绿灯评估）

**风险预警贯穿全程**:
- 早期预警（机会阶段）
- 中期预警（筹建阶段）
- 运营期预警（30+指标监控）
