## 二、动态定价（Dynamic Pricing）

### 2.1 BAR层级体系

**BAR**（Best Available Rate）最优可售房价体系：

```
门市价 (Rack Rate)
    │
    ├─ 提前预订价（7天/14天/21天）
    │
    ├─ 会员专享价
    │
    ├─ 协议企业价
    │
    ├─ 团队/会议价
    │
    └─ 促销/尾房价
```

**价格层级设计原则**:
- 门市价作为价格锚点（ rarely sold ）
- 提前预订价引导提前规划
- 会员价提升忠诚度
- 协议价锁定稳定客源

### 2.2 价格弹性调整策略

**需求定价矩阵**:

| 需求强度 | 价格策略 | ADR调整 | Occ预期 | 适用场景 |
|----------|----------|---------|---------|----------|
| **超高需求** | 溢价定价 | +50~200% | 95%+ | 重大展会、演唱会 |
| **高需求** | 标准BAR | 基准价 | 85-95% | 旺季周末 |
| **平需求** | 灵活调整 | ±10% | 70-85% | 平季工作日 |
| **低需求** | 促销定价 | -20% | 60-70% | 淡季引流 |
| **超低需求** | 尾房抛售 | -30% | 填充率优先 | 当日库存 |

### 2.3 竞争定价监控

**竞争群设定**:
- 3-5家同档次/同区域竞品
- 定期价格扫描（建议每日）
- 价格差距监控（±15%警戒线）

## 四、库存控制（Inventory Control）

### 4.1 房型差价策略（Upsell）

**房型梯度设计**:
```
基础房 ──→ 高级房 ──→ 豪华房 ──→ 套房
 +¥50      +¥100       +¥200
  (升级率目标30%)
```

**Upsell技巧**:
- 前台话术培训
- 会员免费升级权益
- 差价可视化展示

### 4.2 LOS限制（Length of Stay）

**入住长度控制**:
| 场景 | LOS策略 | 目的 |
|------|---------|------|
| 高需求日 | 最小2晚 | 防止低价预订占用高价值日 |
| 低需求日 | 最大1晚 | 保留库存给潜在长住客 |
| 周末 | 周末套餐（2晚） | 提升周末入住率 |

### 4.3 超售策略（Overbooking）

**超售计算**:
```
超售房间数 = 预测No-show率 × 可售房数

参考No-show率:
- 预付订单：5%
- 担保订单：10%
- 普通预订：15%
```

**超售风险管控**:
- 实时库存监控
- 备选方案（附近酒店）
- 补偿标准（升级/赔偿）

## 五、需求预测（Forecasting）

### 5.1 预测模型

**历史数据基准**:
- 去年同期数据
- 过去4周趋势
- 过去3个月平均

**影响因素**:
- 节假日/周末
- 展会/活动日历
- 竞品动态
- 宏观经济

### 5.2 预测精度管理

| 预测周期 | 误差容忍 | 调整频率 |
|----------|----------|----------|
| 90天远期 | ±15% | 每周 |
| 30天近期 | ±10% | 每日 |
| 7天短期 | ±5% | 实时 |

## 六、运营标准体系

### 6.1 前台服务标准

**入住流程**:
| 步骤 | 标准 | 时限 |
|------|------|------|
| 迎接问候 | 3米微笑、1米问候 | 即时 |
| 证件登记 | 身份证读取 | 1分钟 |
| 房型确认 | 确认房型/床型/特殊需求 | 1分钟 |
| 支付押金 | 预授权/现金 | 1分钟 |
| 房卡制作 | 制作/测试 | 30秒 |
| 行李指引 | 电梯/房间方向 | 30秒 |
| **总计** | | **3-4分钟** |

**退房流程（0秒退房）**:
- 预授权自动解冻
- 电子账单推送
- 房卡自动注销
- 无需前台排队

### 6.2 客房服务标准

**清洁检查点（72项）**:
| 区域 | 检查项数 | 关键项 |
|------|----------|--------|
| 卧室 | 25项 | 床品、地面、灰尘 |
| 卫生间 | 20项 | 水渍、毛发、消毒 |
|  minibar | 10项 | 库存、保质期 |
| 设施 | 17项 | 灯光、空调、电视 |

**铺床16步标准**（此处省略详细步骤）

### 6.3 响应时效标准

| 需求类型 | 响应时间 | 完成时间 |
|----------|----------|----------|
| 客房清洁 | 即时 | 退房后30分钟 |
| 维修报修 | 15分钟 | 2小时 |
| 物品借用 | 15分钟 | 30分钟 |
| 投诉处理 | 即时 | 24小时闭环 |
| 叫醒服务 | 预约时间 | 误差±2分钟 |

## 七、收益管理会议机制

### 7.1 日常会议

| 会议 | 频率 | 参与者 | 议题 |
|------|------|--------|------|
| 晨会 | 每日 | 运营核心 | 当日库存、价格调整 |
| 周会 | 每周 | 部门经理 | 本周预测、下周策略 |
| 月会 | 每月 | 管理层 | 月度复盘、竞争分析 |

### 7.2 关键决策节点

```
T-90天：季度预测、大型活动定价
T-30天：月度计划、价格策略确定
T-7天：  周度调整、库存精细管理
T-1天：  当日决策、尾房处理
```
