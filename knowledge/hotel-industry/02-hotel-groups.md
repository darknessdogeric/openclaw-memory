## 四、选择决策框架

### 4.1 投资决策矩阵

```
                    高投资额
                       │
        ┌──────────────┼──────────────┐
        │   高端品牌    │   奢华品牌    │
        │  (万豪/洲际)  │  (丽思/W)    │
        │              │              │
 高品牌  ├──────────────┼──────────────┤  高品牌
 溢价需求│  中高端品牌   │   国际中端    │ 溢价需求
        │ (美居/全季)  │ (智选/欢朋)  │
        │              │              │
        └──────────────┼──────────────┘
        低品牌溢价需求 │ 低品牌溢价需求
                       │
                    低投资额
```

### 4.2 选择标准检查清单

**投资预算匹配**:
- [ ] 单房投资预算？（经济型6-8万/中端10-12万/高端15万+）
- [ ] 品牌加盟费可承受？
- [ ] 持续管理费用占比？

**目标客群匹配**:
- [ ] 物业位置适合的客群？
- [ ] 品牌调性与客群契合？
- [ ] 竞品品牌分布情况？

**地域竞争分析**:
- [ ] 3公里内同品牌数量？
- [ ] 竞品ADR水平？
- [ ] 品牌保护半径？

**运营支持评估**:
- [ ] 中央预订系统贡献？
- [ ] 培训体系完善度？
- [ ] 供应链集采优势？

### 4.3 品牌谈判要点

**费用谈判**:
- 初始加盟费分期支付
- 管理费阶梯优惠（前6个月减免）
- 系统费包干或上限

**合同条款**:
- 品牌保护半径（建议3-5公里）
- 合同期限与续约条件
- 退出机制与违约成本

**支持承诺**:
- 开业前支持天数
- 总经理派驻时长
- 营销资源投入
