## 二、关键决策节点checklist

### 节点1：初筛决策（第一阶段后）

**必须全部满足才能进入第二阶段**:
- [ ] 城市级别符合投资战略（一线/新一线/二线）
- [ ] 物业位置可视性良好（主干道可见）
- [ ] 交通可达性达标（地铁<800m或有停车场）
- [ ] 投资额在预算范围内（±20%）
- [ ] 无重大法律瑕疵（产权清晰）
- [ ] 时间窗口可配合（筹建期无冲突）

**加分项**:
- [ ] 区域供应紧张（供需比<0.8）
- [ ] 政策支持（旅游/商务/会展）
- [ ] 独特资源（景观/历史/文化）
- [ ] 业主有合作意愿（租金优惠/长合约）

**一票否决项**（直接放弃）:
- [ ] 产权纠纷或抵押未解除
- [ ] 规划用途不符（无法办酒店证照）
- [ ] 重大安全隐患（结构/消防）
- [ ] 周边环境恶劣（污染/噪音/治安）

### 节点2：可行性决策（第三阶段后）

**定量指标**:
| 指标 | 通过标准 | 数据来源 |
|------|----------|----------|
| 城市GDP增速 | >全国平均 | 统计局 |
| 供需比 | <1.2 | 26号文件计算 |
| 竞品RevPAR | 可支撑目标ADR | STR/OTA |
| 物业评分 | >3.5/5分 | 26号文件评估表 |
| 预测回收期 | <6年 | 财务测算 |
| 悲观情景IRR | >10% | 敏感性分析 |

**定性评估**:
- [ ] 品牌方可入驻（有适配品牌）
- [ ] 团队可组建（GM候选人已接触）
- [ ] 资金来源确定（自有资金/融资/合作方）
- [ ] 退出路径清晰（未来买家类型已识别）

### 节点3：投资决议（第四阶段后）

**投资决策委员会材料清单**:
1. 投资建议书（项目概况/市场分析/财务测算）
2. 可行性研究报告（26号文件方法论）
3. 财务模型（Excel，含敏感性分析）
4. 尽职调查报告（法律/财务/商业）
5. 风险分析报告（风险点+应对措施）
6. 投资协议草案（关键条款摘要）

**决策标准**:
- 全票通过：立即执行
- 有条件通过：修改后重新提交
- 否决：归档或6个月后重提

### 节点4：交割前核查（第六阶段后）

**资金核查**:
- [ ] 投资款已到位（或融资承诺函）
- [ ] 品牌方已签署合同
- [ ] 物业租约/购买合同已签署
- [ ] 共管账户已设立（如需要）

**证照核查**:
- [ ] 营业执照预核准
- [ ] 特种行业许可证可办理
- [ ] 消防设计审核通过
- [ ] 环评批复（如需要）

### 节点5：退出决策（第八阶段后）

**退出时机评估表**（详见11号文件）:
| 维度 | 绿灯（积极退出） | 黄灯（择机退出） | 红灯（暂缓退出） |
|------|------------------|------------------|------------------|
| 经济周期 | 繁荣期 | 复苏期 | 衰退/萧条期 |
| 行业景气 | RevPAR创新高 | 稳定 | 连续下滑 |
| 资本市场 | 活跃，买家多 | 一般 | 冷清 |
| 资产状态 | 运营成熟期 | 爬坡期 | 改造期 |
| 财务表现 | GOP率高位 | 达标 | 低于预算 |

## 三、风险预警信号

### 早期预警信号（机会阶段）

**信息类风险**:
- ⚠️ 信息来源单一（中介独家）
- ⚠️ 业主急于出手（低于市场价20%+）
- ⚠️ 隐瞒关键信息（租约剩余年限/抵押情况）

**市场类风险**:
- ⚠️ 3公里内同品牌已有多家
- ⚠️ 主要客源企业搬迁/倒闭传闻
- ⚠️ 城市规划重大调整（道路改线/地铁取消）

### 中期预警信号（筹建阶段）

**成本类风险**:
- 🚨 改造成本超预算>20%
- 🚨 设计变更频繁（>5次重大变更）
- 🚨 施工进度滞后>3个月

**合规类风险**:
- 🚨 消防验收不通过需重大整改
- 🚨 特种行业许可证办理受阻
- 🚨 环保投诉导致停工

### 运营期预警信号（投后管理）

**经营类风险**:
- 🔴 连续6个月入住率<60%
- 🔴 RevPAR低于竞品>30%
- 🔴 GOP率低于预算>10个百分点
- 🔴 现金流为负持续>6个月

**管理类风险**:
- 🔴 GM频繁更换（1年>2次）
- 🔴 核心团队集体离职
- 🔴 重大安全事故/舆情危机

**外部类风险**:
- 🔴 区域竞争恶化（新增供给>50%）
- 🔴 主要协议客户流失>30%
- 🔴 租约到期业主大幅涨租（>50%）

| **区域竞争度** | 20% | ___ | ___ | 供需比/竞争格局/进入壁垒 |
| **物业条件** | 20% | ___ | ___ | 位置/可视性/租约/改造难度 |
| **财务可行性** | 25% | ___ | ___ | 回收期/IRR/敏感性/现金流 |
| **品牌适配度** | 10% | ___ | ___ | 可选品牌/费用/支持能力 |
| **团队能力** | 10% | ___ | ___ | GM候选人/业主经验/资源 |
| **总分** | 100% | - | ___ | |

**决策标准**:
- ≥80分：优质项目，积极推进
- 60-80分：可投项目，关注风险点
- <60分：不建议投资

```
            └─ 是 → 投委会批准？
                 ├─ 否 → 补充材料或放弃
                 └─ 是 → 投资决策通过，执行投资
```

### 工具3：风险矩阵

| 风险类型 | 高概率/高影响 | 高概率/低影响 | 低概率/高影响 | 低概率/低影响 |
|----------|---------------|---------------|---------------|---------------|
| **市场风险** | 竞争恶化 | 季节性波动 | 经济衰退 | 政策微调 |
| **运营风险** | GM能力不足 | 员工流失 | 重大事故 | 设备小故障 |
| **财务风险** | 成本超支 | 汇率波动 | 利率大幅上升 | 通胀温和 |
| **法律风险** | 租约纠纷 | 证照延期 | 产权诉讼 | 合同纠纷 |

**应对策略**:
- 高概率/高影响：规避或转移（保险/对赌）
- 高概率/低影响：缓解（流程优化）
- 低概率/高影响：应急计划
- 低概率/低影响：接受
