## 二、消防安全管理

### 2.1 消防安全责任制

**三级责任体系**:
```
第一责任人：总经理
├── 全面负责酒店消防安全
├── 提供资源保障
└── 签署安全责任书

第二责任人：安保经理
├── 日常消防管理
├── 制度制定与培训
└── 与政府对接

直接责任人：各部门
├── 部门内部防火检查
├── 员工消防培训
└── 隐患整改
```

### 2.2 消防设施管理

**设施清单**:
| 设施 | 数量标准 | 检查频率 |
|------|----------|----------|
| 灭火器 | 50㎡/具 | 每月 |
| 消火栓 | 50米/个 | 每月 |
| 烟感 | 全覆盖 | 每季度测试 |
| 喷淋 | 全覆盖 | 每季度检查 |
| 应急灯 | 全覆盖 | 每月 |
| 疏散指示 | 全覆盖 | 每月 |

**管理要求**:
- 设施完好率100%
- 不被遮挡、挪用
- 定期检查记录
- 及时维修更换

### 2.3 火灾预防重点

**高风险区域**:
| 区域 | 风险 | 防控措施 |
|------|------|----------|
| 厨房 | 油烟/燃气 | 油烟清洗、燃气报警 |
| 客房 | 电器/吸烟 | 电器检查、禁烟提示 |
| 配电室 | 电气火灾 | 温度监测、定期检测 |
| 仓库 | 可燃物 | 分类存放、禁火管理 |

### 2.4 消防培训与演练

**培训内容**:
- 消防知识（入职必训）
- 器材使用（实操）
- 疏散程序
- 应急报告

**演练频率**:
- 灭火演练：每半年
- 疏散演练：每半年
- 联合演练：每年（与消防队）

## 三、治安防范管理

### 3.1 实名登记制度

**《旅馆业治安管理办法》要求**:
- 一人一证登记
- 证件真伪核验
- 信息实时上传
- 访客登记

**执行要点**:
- 证件识别仪配置
- 人脸识别比对（如有）
- 异常情况报告
- 信息保密

### 3.2 访客管理

**流程**:
1. 访客登记（身份证）
2. 联系住客确认
3. 发放访客卡
4. 限时离开（23:00前）

### 3.3 贵重物品保管

**保险箱服务**:
- 前台免费保管
- 使用登记
- 双人开启
- 监控覆盖

### 3.4 停车场管理

**安全要求**:
- 24小时监控
- 进出记录
- 巡逻检查
- 车辆安全提示

## 五、应急预案

### 5.1 应急预案体系

| 预案类型 | 适用场景 | 启动条件 |
|----------|----------|----------|
| 火灾预案 | 火灾事故 | 火警确认 |
| 治安预案 | 刑事案件 | 报警/发现 |
| 食品预案 | 食物中毒 | 疑似病例 |
| 停电预案 | 电力中断 | 停电发生 |
| 电梯困人 | 电梯故障 | 困人报警 |
| 防汛预案 | 暴雨内涝 | 预警发布 |
| 防疫预案 | 传染病 | 疫情通报 |

### 5.2 火灾应急预案

**应急流程**:
```
发现火情
   ↓
确认火情（真假火警）
   ↓
初起灭火（灭火器/消火栓）
   ↓
报警（119/内部报警）
   ↓
组织疏散
   ↓
清点人数
   ↓
配合消防队
   ↓
善后处理
```

**疏散要点**:
- 按疏散预案路线
- 优先客人后员工
- 禁用电梯
- 集合点清点
- 禁止返回

### 5.3 应急物资配置

| 物资 | 配置位置 | 数量 |
|------|----------|------|
| 灭火器 | 公共区域、厨房、机房 | 按标准 |
| 消防斧 | 消防控制室 | 1-2把 |
| 应急灯 | 各楼层 | 按标准 |
| 扩音器 | 前台、消防控制室 | 2个 |
| 急救箱 | 前台 | 1个 |
| 防汛物资 | 地下室入口 | 沙袋、挡板等 |
