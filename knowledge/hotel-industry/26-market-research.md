## 二、第一级：城市层面调研（宏观）

### 2.1 调研维度

| 维度 | 调研内容 | 数据来源 | 判断标准 |
|------|----------|----------|----------|
| **经济数据** | GDP/人均收入/三产结构 | 统计局/政府工作报告 | 三产占比>50%为佳 |
| **人口流动** | 常住人口/流动人口/商务人口 | 人口普查/移动大数据 | 净流入城市优先 |
| **产业结构** | 支柱产业/企业数量/展会活动 | 工商局/会展办 | 商旅需求旺盛 |
| **交通基建** | 机场/高铁/地铁/高速 | 交通局/12306 | 交通枢纽城市溢价 |
| **旅游热度** | 游客量/景区数量/旅游收入 | 文旅局 | 过夜游客比例 |
| **酒店供需** | 酒店数量/客房量/连锁化率 | STR/饭店协会 | 连锁化率<30%有机会 |
| **政策环境** | 招商引资/旅游政策/规划 | 政府文件 | 政策支持力度 |

### 2.2 城市分级投资指引

| 城市级别 | 经济指标 | 酒店市场特征 | 投资建议 | 适配品牌 |
|----------|----------|--------------|----------|----------|
| **一线** | GDP>3万亿 | 饱和/存量博弈 | 改造升级/精品 | 国际奢华/高端 |
| **新一线** | GDP 1-3万亿 | 增长/竞争激烈 | 中端/中高端 | 国际中高端/本土头部 |
| **二线** | GDP 5k-1万亿 | 机会/蓝海市场 | 中端/精选 | 本土中端/区域品牌 |
| **三线** | GDP<5k亿 | 分散/下沉机会 | 经济型/中端 | 连锁经济型/轻中端 |

### 2.3 城市进入决策树

```
城市GDP增速>6%？
  ├─ 否 → 谨慎进入
  └─ 是 → 三产占比>45%？
           ├─ 否 → 工业城市，选商务型
           └─ 是 → 净流入人口>10万？
                    ├─ 否 → 旅游城市，季节性风险
                    └─ 是 → 连锁化率<35%？
                             ├─ 否 → 竞争激烈，需差异化
                             └─ 是 → 优质市场，可进入
```

## 三、第二级：区域层面调研（中观）

### 3.1 商圈类型分析

| 商圈类型 | 客群特征 |  weekday表现 |  weekend表现 | 淡旺季 | 代表位置 |
|----------|----------|-------------|-------------|--------|----------|
| **CBD商务** | 商务客 | 强 | 弱 | 不明显 | 市中心/金融区 |
| **交通枢纽** | 中转客 | 稳定 | 稳定 | 不明显 | 机场/高铁/地铁口 |
| **景区周边** | 休闲客 | 弱 | 强 | 明显 | 景点1-3公里 |
| **大学城** | 学生/家长 | 弱 | 中 | 寒暑假淡 | 高校集中区 |
| **工业园** | 商务/务工 | 中 | 弱 | 春节淡 | 开发区/产业园区 |
| **老城区** | 混合客群 | 中 | 中 | 不明显 | 传统市中心 |
| **新区** | 规划客群 | 视发展阶段 | 视发展阶段 | 培育期长 | 新城/新区 |

### 3.2 竞争分析五维模型

```
竞争分析雷达图（5分制）

价格竞争 ←─────────→ 服务竞争
    │                   │
    │    你的位置       │
    │        ★          │
    │                   │
品牌竞争 ←─────────→ 位置竞争
         │
         ↓
      渠道竞争
```

**调研要素**:

| 维度 | 调研内容 | 方法 | 输出 |
|------|----------|------|------|
| **供给量** | 3公里内酒店数量/客房量 | 实地踩盘+OTA地图 | 供需比 |
| **价格带** | 各档次ADR/Occ分布 | STR数据/爬虫 | 价格梯度 |
| **品牌分布** | 国际/本土/单体占比 | 品牌官网+OTA | 竞争格局 |
| **渠道结构** | OTA/直销/协议占比 | 前台暗访 | 渠道策略 |
| **服务质量** | 点评分数/差评关键词 | 爬虫+文本分析 | 服务差距 |
| **设施水平** | 装修年限/硬件设施 | 实地体验 | 硬件差距 |

### 3.3 竞争强度评估

**供需比计算**:
```
供需比 = 区域客房供给量 / 区域住宿需求量

需求量估算 = 
  商务需求（写字楼面积×0.1人/㎡×10%外宿率）
+ 旅游需求（景区年游客量×10%过夜率/365）
+ 会议需求（会展面积×估算）
+ 交通需求（机场吞吐量×1%过夜率）

判断标准：
- <0.8：供不应求，可进入
- 0.8-1.2：供需平衡，需差异化
- >1.2：供过于求，谨慎进入
```

## 四、第三级：周边层面调研（微观）

### 4.1 物业条件评估表

| 评估项 | 权重 | 评估标准 | 评分 |
|--------|------|----------|------|
| **可视性** | 15% | 主干道可见/有招牌位 | 0-5分 |
| **可达性** | 20% | 多入口/停车方便/地铁<500m | 0-5分 |
| **周边业态** | 15% | 配套完善/无负面业态 | 0-5分 |
| **建筑结构** | 15% | 层高>3m/柱距>8m/无暗房 | 0-5分 |
| **面积效率** | 10% | 得房率>70%/出房率>60% | 0-5分 |
| **租约条件** | 15% | 租期>10年/租金递增<5%/无拆迁风险 | 0-5分 |
| **改造空间** | 10% | 装修自由度/消防可达性 | 0-5分 |

**评分标准**:
- 4.5-5分：优质物业，可溢价投资
- 3.5-4.5分：标准物业，正常投资
- 2.5-3.5分：有缺陷，谨慎投资/降低单房投资
- <2.5分：不建议投资

### 4.2 客流量实测方法

**方法一：人流计数（连续3天）**
- 时段：7-9点/12-14点/18-20点
- 工具：计数器APP
- 记录：人流量/提行李比例/年龄分布

**方法二：车流观察**
- 观察出租车/网约车上下客频率
- 观察停车周转率
- 观察车牌属地（本地/外地比例）

**方法三：竞品蹲点**
- 观察竞品酒店大堂人流
- 记录入住高峰时段
- 估算实际入住率

**方法四：商户访谈**
- 访谈周边餐饮/便利店
- 了解客源类型
- 了解淡旺季节律

### 4.3 噪音与干扰因素

**负面因素清单**:
- [ ] 临近高架/高速（噪音）
- [ ] 临近夜市/酒吧街（噪音+安全）
- [ ] 临近医院（特定客群）
- [ ] 临近殡仪馆（忌讳）
- [ ] 周边拆迁/施工（短期）
- [ ] 物业有法律纠纷（风险）

## 五、投资额度测算模型

### 5.1 单房投资基准

| 档次 | 单房投资（万元/间） | 总投资（100间） | 回收期目标 |
|------|-------------------|----------------|-----------|
| 经济型 | 6-8 | 600-800万 | 3-4年 |
| 中端 | 10-12 | 1000-1200万 | 4-6年 |
| 中高端 | 13-16 | 1300-1600万 | 5-7年 |
| 高端 | 18-25 | 1800-2500万 | 6-8年 |
| 奢华 | 30+ | 3000万+ | 8-10年 |

### 5.2 动态调整因子

**上调因子**（+10-30%）:
- 一线城市核心商圈：+20%
- 老旧物业改造成本高：+15%
- 物业结构复杂：+10%
- 高端定制装修：+25%

**下调因子**（-10-20%）:
- 物业已部分装修：-15%
- 批量采购优势：-10%
- 集团集采：-15%
- 轻改造模式：-20%

### 5.3 总投资构成

```
总投资
├── 营建成本（70-75%）
│   ├── 土建改造（10-15%）
│   ├── 装修工程（40-50%）
│   ├── 机电设备（15-20%）
│   └── FF&E家具软装（10-15%）
├── 开办费（10-15%）
│   ├── 前期费用（证照/设计/招标）
│   ├── 系统上线（PMS/门锁/网络）
│   ├── 开业筹备（人员/培训/营销）
│   └── 流动资金（3个月运营资金）
└── 不可预见费（10-15%）
    └── 变更/涨价/延期成本
```

### 5.4 RevPAR预测模型

**基础公式**:
```
预测RevPAR = 竞品RevPAR × 位置系数 × 品牌系数 × 产品系数

位置系数：
- 核心商圈：1.1-1.2
- 次级商圈：0.9-1.0
- 边缘位置：0.8-0.9

品牌系数：
- 国际品牌：1.1-1.3
- 本土头部：1.0-1.1
- 区域品牌：0.9-1.0
- 单体/自创：0.8-0.9

产品系数：
- 新装修+设计亮点：1.1-1.2
- 标准产品：1.0
- 老旧/无特色：0.8-0.9
```

## 六、财务测算模型

### 6.1 收入预测

```
年营业收入 = RevPAR × 365 × 房间数

示例：
- 房间数：100间
- 预测RevPAR：¥320
- 年收入 = 320 × 365 × 100 = ¥1,168万

收入结构：
├── 客房收入（65-75%）
├── 餐饮收入（15-25%）
└── 其他收入（5-10%）
```

### 6.2 成本预测

```
年运营成本
├── 固定成本（55-65%）
│   ├── 人力成本（25-30%）
│   ├── 租金/折旧（20-30%）
│   ├── 能耗（8-12%）
│   └── 维保/其他（3-5%）
└── 变动成本（25-35%）
    ├── 餐饮成本（8-12%）
    ├── 客房用品（2-3%）
    ├── 营销费用（5-8%）
    └── OTA佣金（3-8%）

目标GOP率：
- 经济型：40-45%
- 中端：35-40%
- 中高端：30-35%
- 高端：25-30%
```

### 6.3 投资回报测算

```
年GOP = 年营业收入 × GOP率

静态回收期 = 总投资 / 年GOP

动态回收期（考虑折现率8%）：
NPV = ∑(第t年现金流 / (1+8%)^t) - 总投资

IRR = 使NPV=0的折现率
```

### 6.4 敏感性分析

| 变量 | 基准 | 悲观(-20%) | 乐观(+20%) | 影响程度 |
|------|------|-----------|-----------|---------|
| RevPAR | ¥320 | ¥256 | ¥384 | 高 |
| Occ | 80% | 64% | 96% | 高 |
| GOP率 | 38% | 30% | 46% | 中 |
| 投资额 | 1200万 | 1440万 | 960万 | 中 |

**压力测试**:
- 悲观情景：回收期延长50%，是否可接受？
- 极端情景：亏损耐受期有多久？

## 七、品牌适配决策

### 7.1 品牌选择决策树

```
投资预算？
├─ <800万 → 经济型/自创品牌
│           ├─ 有无酒店经验？
│           │   ├─ 无 → 加盟连锁（如家/汉庭）
│           │   └─ 有 → 可自创品牌
├─ 800-1500万 → 中端/中高端
│                ├─ 物业位置？
│                │   ├─ 核心商圈 → 国际品牌（美居/假日）
│                │   ├─ 次级商圈 → 本土品牌（全季/桔子）
│                │   └─ 特色物业 → 精品品牌（亚朵/自创）
└─ >1500万 → 高端/奢华
             ├─ 城市级别？
             │   ├─ 一线/新一线 → 国际品牌（万豪/洲际）
             │   └─ 二线 → 本土高端/精品
```

### 7.2 自创品牌 vs 加盟品牌

| 维度 | 自创品牌 | 加盟品牌 |
|------|----------|----------|
| **前期投入** | 低（无加盟费） | 高（加盟费+系统费） |
| **品牌认知** | 需从零建立 | 直接使用 |
| **获客成本** | 高（需自建渠道） | 低（会员输送） |
| **运营支持** | 自主摸索 | 总部支持 |
| **供应链** | 自主采购 | 集采优势 |
| **失败风险** | 高 | 相对较低 |
| **适用场景** | 特色物业/有经验/预算紧 | 标准化物业/求稳/新手 |

**自创品牌条件**:
- 投资预算紧张（可省3-5万/房加盟费）
- 有独特物业（历史建筑/景观资源）
- 投资人有酒店运营经验
- 区域市场无合适品牌
- 有强营销能力

### 7.3 品牌费用对比（中端为例）

| 品牌 | 加盟费 | 管理费 | 系统费 | 单房投资 | 适合投资人 |
|------|--------|--------|--------|----------|------------|
| **美居** | 5-8k/房 | 5-6% | 1% | 12-15万 | 追求国际品牌背书 |
| **全季** | 4-6k/房 | 5% | 0.5% | 10-12万 | 追求效率与规模 |
| **桔子** | 3.5-5.5k/房 | 5% | 0.5% | 9-11万 | 追求设计感 |
| **亚朵** | 6-10k/房 | 6% | 1% | 13-16万 | 追求生活方式 |
| **维也纳** | 3-5k/房 | 4-6% | 0.5% | 8-10万 | 追求投资回报 |

## 八、合作模式选择

### 8.1 五种合作模式对比

| 模式 | 资产归属 | 管理方 | 风险 | 收益 | 适用场景 |
|------|----------|--------|------|------|----------|
| **直营** | 品牌方 | 品牌方 | 高 | 全收益 | 战略要地/旗舰店 |
| **委托管理** | 业主 | 品牌方重管理 | 低 | 管理费 | 高端/业主缺经验 |
| **特许经营** | 业主 | 业主自主 | 中 | 全收益-加盟费 | 有经验投资人 |
| **租赁经营** | 第三方 | 品牌方 | 中高 | 全收益-租金 | 城市核心 |
| **轻资产联盟** | 业主 | 平台方 | 极低 | 服务费 | 单体酒店/民宿 |

### 8.2 模式选择矩阵

```
业主经验？
├─ 丰富 → 特许经营/自创品牌
│          └─ 追求稳定 → 特许（国际品牌）
│          └─ 追求灵活 → 自创品牌
├─ 一般 → 委托管理/特许经营
│          └─ 高端定位 → 委托管理
│          └─ 中端定位 → 特许经营
└─ 无 → 委托管理/加盟
           └─ 高预算 → 委托管理
           └─ 有限预算 → 加盟
```

### 8.3 合同谈判要点

**委托管理合同**:
- 基本管理费：营收的3-5%
- 激励管理费：GOP超额部分的8-12%
- 合同期限：10-15年
- 业绩对赌：低于预算的处理
- 业主审批权：多大额度以上需审批
- 退出条款：提前解约条件

**特许经营合同**:
- 加盟费：一次性/按房
- 特许权使用费：营收的4-6%
- 营销费：营收的2-3%
- 系统费：固定/按营收
- 品牌保护：保护半径/年限
- 装修标准：改造频率/标准
- 中央预订：贡献度保证

## 九、供应商与合作方匹配

### 9.1 按投资规模匹配

**小型投资（<1000万）**:
- PMS：别样红/云掌柜（低成本）
- 布草：区域供应商（价格低）
- 装修：本地施工队（灵活）
- 营销：自主运营+代运营

**中型投资（1000-3000万）**:
- PMS：绿云/西软（中端主流）
- 布草：南通/淮安专业厂
- 装修：连锁推荐施工商
- 营销：品牌方支持+本地代理

**大型投资（>3000万）**:
- PMS：Opera/定制开发
- 布草：康乃馨/斯得福（高端）
- 装修：知名设计公司+总包
- 营销：4A公司+品牌联合

### 9.2 按品牌档次匹配

| 档次 | PMS | 设计 | 布草 | 智能客控 |
|------|-----|------|------|----------|
| 经济型 | 别样红 | 标准化设计 | 经济型布草 | 基础门锁 |
| 中端 | 绿云/西软 | 品牌标准设计 | 中端布草 | 小米/涂鸦 |
| 中高端 | 西软/Opera | 设计师事务所 | 高端布草 | 肯特/华为 |
| 奢华 | Opera/定制 | 国际设计（HBA/Wilson） | 进口/定制 | 全屋智能 |

### 9.3 关键供应商选择

**PMS系统**:
- 中端：绿云（云化/接口开放）
- 高端：Opera（功能全面）
- 经济型：别样红（美团生态）

**设计公司**:
- 国际：HBA/Wilson/Yabu
- 国内：金螳螂/亚厦/CCD
- 精品：独立设计师

**施工总包**:
- 连锁酒店：品牌推荐/认证施工商
- 独立项目：本地龙头装饰公司
- 需考察：酒店项目经验/资金实力

**OTA合作**:
- 主力：携程（流量最大）
